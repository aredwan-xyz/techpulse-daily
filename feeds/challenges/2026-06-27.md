TITLE: Balancing Parentheses with Minimum Swaps
DIFFICULTY: Hard
TOPIC: Stack & Queue
ESTIMATED_TIME: 45 minutes

PROBLEM:
You are given a string that contains a mixture of parentheses - including '(', ')', '{', '}', '[' and ']'. Your task is to determine the minimum number of swaps needed to balance the parentheses in the string. A string is considered balanced if:

1. Every opening bracket has a corresponding and correctly positioned closing bracket.
2. The brackets are closed in the reverse order of their opening.

You can swap any two characters in the string, and your goal is to compute the fewest number of such swaps necessary to achieve a balanced string.

EXAMPLES:
Input: "{[()]}"
Output: 0
Explanation: The string is already balanced.

Input: "{[(]}"
Output: 1
Explanation: One possible solution is to swap the second '(' with the last ']', resulting in a balanced string: "{[()]}" or "{[()}].

Input: "}}"
Output: 1
Explanation: One swap will make it balanced, e.g. by changing it to "{}".

CONSTRAINTS:
- The input string will have a length n where 0 ≤ n ≤ 10^5.
- The string will only contain the characters '(', ')', '{', '}', '[' and ']'.

HINT:
Try to utilize a stack to help track the balance of the parentheses as you iterate through the string.

SOLUTION_PYTHON:
```python
def min_swaps_to_balance(s: str) -> int:
    # Use a stack to maintain the balance of parentheses
    stack = []
    open_count = 0
    close_count = 0

    for char in s:
        if char in '{[(':
            stack.append(char)
            open_count += 1
        elif char in '}])':
            if stack:
                top = stack[-1]
                if (top == '{' and char == '}') or (top == '[' and char == ']') or (top == '(' and char == ')'):
                    stack.pop()  # Valid match found
                else:
                    close_count += 1  # Invalid closing
            else:
                close_count += 1  # Extra closing bracket

    # To balance the string, half of the mismatched count must be swapped
    return (open_count + close_count) // 2

# Example usage
print(min_swaps_to_balance("{[(]}"))  # Output: 1
```

SOLUTION_JS:
```javascript
function minSwapsToBalance(s) {
    const stack = [];
    let openCount = 0;
    let closeCount = 0;

    for (const char of s) {
        if (['{', '[', '('].includes(char)) {
            stack.push(char);
            openCount++;
        } else if (['}', ']', ')'].includes(char)) {
            if (stack.length > 0) {
                const top = stack[stack.length - 1];
                if ((top === '{' && char === '}') || (top === '[' && char === ']') || (top === '(' && char === ')')) {
                    stack.pop(); // Valid match found
                } else {
                    closeCount++; // Invalid closing
                }
            } else {
                closeCount++; // Extra closing bracket
            }
        }
    }

    // To balance the string, half of the mismatched count must be swapped
    return Math.floor((openCount + closeCount) / 2);
}

// Example usage
console.log(minSwapsToBalance("{[(]}")); // Output: 1
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The approach relies on a stack to keep track of the opening brackets. As we traverse through the string, we count how many valid closing brackets match the opening ones. The unmatched brackets help figure out how many swaps are needed to balance the string. Each swap can fix two unmatched brackets, thus we take the total of unmatched opening and closing brackets divided by two to get the minimum swaps.
