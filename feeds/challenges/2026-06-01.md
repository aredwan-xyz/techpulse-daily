TITLE: Overlapping Bits Checker  
DIFFICULTY: Easy  
TOPIC: Bit Manipulation  
ESTIMATED_TIME: 15 minutes  

PROBLEM:  
You are given two integers, `a` and `b`. Your task is to determine if there are any overlapping bits set to `1` in their binary representations. Specifically, return `True` if there is at least one bit position where both `a` and `b` have a `1`; otherwise, return `False`. This can be useful in various applications where you need to avoid conflicts in bit flags.

EXAMPLES:  
Input: `a = 5, b = 3`  
Output: `True`  
Explanation: The binary representations are `101` (for 5) and `011` (for 3). Both have a `1` in the second bit position (from the right).

Input: `a = 4, b = 2`  
Output: `False`  
Explanation: The binary representations are `100` (for 4) and `010` (for 2). There are no overlapping `1` bits.

CONSTRAINTS:  
- `0 <= a, b <= 2^31 - 1`  
- Both `a` and `b` are non-negative integers.

HINT:  
Consider using the bitwise AND operator to find overlapping bits.

SOLUTION_PYTHON:
```python
def has_overlapping_bits(a: int, b: int) -> bool:
    # Use the bitwise AND operator to check if any bits overlap
    return (a & b) != 0

# Example usage
print(has_overlapping_bits(5, 3))  # Output: True
print(has_overlapping_bits(4, 2))  # Output: False
```

SOLUTION_JS:
```javascript
function hasOverlappingBits(a, b) {
    // Use the bitwise AND operator to check if any bits overlap
    return (a & b) !== 0;
}

// Example usage
console.log(hasOverlappingBits(5, 3));  // Output: true
console.log(hasOverlappingBits(4, 2));  // Output: false
```

TIME_COMPLEXITY: O(1)  
SPACE_COMPLEXITY: O(1)  

EXPLANATION:  
The approach uses the bitwise AND operator (`&`) to check for common bits set to `1` between the two integers. If the result of `a & b` is greater than `0`, it indicates that there is at least one bit set to `1` in the same position for both numbers. This solution is optimal with constant time and space complexity since it operates on fixed-size integers.
