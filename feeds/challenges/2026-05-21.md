TITLE: Count Distinct Products
DIFFICULTY: Medium
TOPIC: Hash Maps & Sets
ESTIMATED_TIME: 30 minutes

PROBLEM:
You are the manager of an online store selling different types of products. Your customers often buy multiple products in a single order. Each product can be represented as a string (for example, "Laptop", "Phone", "Headphones"). Your task is to create a function that counts the number of distinct products bought across all orders. Each order consists of a list of product strings, and you need to ensure that the same product bought multiple times in a single order only counts once.

Write a function `count_distinct_products(orders: List[List[str]]) -> int` that takes a list of orders, where each order is a list of product strings, and returns the total number of distinct products bought.

EXAMPLES:
Input: 
```python
orders = [["Laptop", "Phone", "Laptop"], ["Headphones", "Phone"], ["Laptop", "Charger"]]
```
Output: 
```
5
```
Explanation: The distinct products are "Laptop", "Phone", "Headphones", and "Charger".

Input: 
```python
orders = [["Tablet", "Charger"], ["Tablet", "Tablet"], ["Charger", "Mouse", "Mouse"]]
```
Output: 
```
4
```
Explanation: The distinct products are "Tablet", "Charger", and "Mouse".

CONSTRAINTS:
- The number of orders (1 ≤ orders.length ≤ 1000)
- Each order can contain between 1 and 100 products (1 ≤ order[i].length ≤ 100)
- Each product string has a length between 1 and 20 characters.

HINT:
Consider using a Hash Set to keep track of the distinct products you encounter.

SOLUTION_PYTHON:
```python
def count_distinct_products(orders):
    # Create a set to store distinct products
    distinct_products = set()
    
    # Iterate through each order
    for order in orders:
        # Add each product in the order to the set
        distinct_products.update(order)
    
    # The size of the set represents the number of distinct products
    return len(distinct_products)

# Time Complexity: O(n) where n is the total number of products in all orders.
# Space Complexity: O(m) where m is the number of distinct products.
```

SOLUTION_JS:
```javascript
function countDistinctProducts(orders) {
    // Use a Set to store distinct products
    const distinctProducts = new Set();
    
    // Iterate through each order
    orders.forEach(order => {
        order.forEach(product => {
            distinctProducts.add(product);
        });
    });
    
    // Return the size of the set as the count of distinct products
    return distinctProducts.size;
}

// Time Complexity: O(n) where n is the total number of products in all orders.
// Space Complexity: O(m) where m is the number of distinct products.
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(m)

EXPLANATION:
The approach utilizes a Hash Set to keep track of distinct product names as we iterate through each order. By adding every product from each order to the set, any duplicates are automatically handled since sets do not allow duplicate values. The final count of distinct products is simply the size of the set, which is efficient and optimal for this task.
