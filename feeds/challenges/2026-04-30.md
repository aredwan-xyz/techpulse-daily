TITLE: Valid Word Squares 
DIFFICULTY: Medium
TOPIC: Recursion & Backtracking
ESTIMATED_TIME: 35 minutes

PROBLEM:
You have a list of words, and your task is to determine if you can form a valid word square using all the words. A word square is a special kind of matrix where the input words are arranged in such a way that the words read the same vertically as they do horizontally. For example, given the words ["ball", "anna", "lily", "lira"], you can form the following word square:

```
ball
anna
lily
lira
```

Write a function that returns true if a valid word square can be formed, otherwise return false.

EXAMPLES:
Input: words = ["ball", "anna", "lily", "lira"]
Output: true
Explanation: The words can be arranged to form a valid word square as shown above.

Input: words = ["hello", "world", "hold", "zero"]
Output: false
Explanation: No arrangement of the words can create a valid word square.

CONSTRAINTS:
- A word's length is at least 1 and at most 10.
- The number of words is between 1 and 20.

HINT:
Consider using recursion to build the word square row by row, checking at each level that the words fit the definition of a valid square.

SOLUTION_PYTHON:
```python
def valid_word_square(words):
    n = len(words)
    
    # Early exit if there are no words or the number of words doesn't match the length
    if n == 0 or any(len(word) != n for word in words):
        return False
    
    # Helper function to check for a valid word square during construction
    def is_valid_square(i):
        for j in range(i):
            if words[i][j] != words[j][i]:
                return False
        return True

    # Backtracking to try forming the word square
    def backtrack(i):
        if i == n:
            return True
        for word in words:
            if valid_lengths_for_square(word, i):
                words[i] = word
                if is_valid_square(i) and backtrack(i + 1):
                    return True
                words[i] = ""  # Reset backtrack
        return False

    return backtrack(0)

def valid_lengths_for_square(word, i):
    """Check if a word can fit in the current position of the square."""
    if len(word) != len(words):
        return False
    return True
```

SOLUTION_JS:
```javascript
function validWordSquare(words) {
    const n = words.length;

    // Early exit if there are no words or if some words' lengths don't match n
    if (n === 0 || words.some(word => word.length !== n)) {
        return false;
    }

    const isValidSquare = (i) => {
        for (let j = 0; j < i; j++) {
            if (words[i][j] !== words[j][i]) {
                return false;
            }
        }
        return true;
    }

    const backtrack = (i) => {
        if (i === n) {
            return true;
        }

        for (const word of words) {
            if (validLengthsForSquare(word, i)) {
                words[i] = word;
                if (isValidSquare(i) && backtrack(i + 1)) {
                    return true;
                }
                words[i] = ""; // Reset for backtracking
            }
        }
        return false;
    }

    return backtrack(0);
}

function validLengthsForSquare(word, i) {
    return word.length === words.length;
}
```

TIME_COMPLEXITY: O(n^2 * m)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The approach uses a backtracking technique to construct the word square incrementally. At each level, we validate whether the current word fits in the specific position by checking against previously placed words. The solution explores all possible arrangements while maintaining a valid structure, making it efficient for the constraints provided.
