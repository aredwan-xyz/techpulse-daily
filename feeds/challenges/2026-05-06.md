TITLE: Unique Character Count
DIFFICULTY: Easy
TOPIC: Arrays & Strings
ESTIMATED_TIME: 20 minutes

PROBLEM:
You are given a string that may contain both letters and digits. Your task is to count how many unique alphanumeric characters (both uppercase and lowercase letters count as the same character) are present in the string. Ignore any characters that are neither letters nor digits. Return the count of these unique characters.

EXAMPLES:
Input: "Hello123"
Output: 6
Explanation: The unique characters are 'H', 'e', 'l', 'o', '1', '3'. Note that 'l' appears twice but is only counted once.

Input: "abcABC!@#"
Output: 6
Explanation: The unique characters are 'a', 'b', 'c', 'A', 'B', 'C'. The special characters are ignored.

CONSTRAINTS:
- The string length will be between 1 and 1000 characters.
- The input string will only contain printable ASCII characters.

HINT:
Consider using a set to keep track of unique characters while iterating through the string.

SOLUTION_PYTHON:
```python
def unique_character_count(s: str) -> int:
    unique_chars = set()
    
    for char in s:
        # Normalize case and check if character is alphanumeric
        if char.isalnum():
            unique_chars.add(char.lower())
    
    return len(unique_chars)

# Example usage:
print(unique_character_count("Hello123"))  # Output: 6
print(unique_character_count("abcABC!@#")) # Output: 6
```

SOLUTION_JS:
```javascript
function uniqueCharacterCount(s) {
    const uniqueChars = new Set();
    
    for (let char of s) {
        // Normalize case and check if character is alphanumeric
        if (/[a-zA-Z0-9]/.test(char)) {
            uniqueChars.add(char.toLowerCase());
        }
    }
    
    return uniqueChars.size;
}

// Example usage:
console.log(uniqueCharacterCount("Hello123"));  // Output: 6
console.log(uniqueCharacterCount("abcABC!@#")); // Output: 6
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The approach uses a set to collect unique characters while iterating through the string. By normalizing the case of each character to lowercase (or uppercase), it ensures that 'A' and 'a' are counted as the same character. The time complexity is O(n) where n is the length of the string, as each character is processed once, and the space complexity is also O(n) due to the storage in the set.

FOLLOW_UP:
Modify the function to return both the count of unique characters and a list of those characters in decreasing order of their first appearance in the string.
