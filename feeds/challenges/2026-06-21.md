TITLE: Temperature Threshold
DIFFICULTY: Medium
TOPIC: Binary Search
ESTIMATED_TIME: 30 minutes

PROBLEM:
A weather monitoring company has collected hourly temperature data for a week and is looking to find optimal days for outdoor activities based on a specified temperature threshold. Your task is to write a function that takes in a list of daily average temperatures and a target temperature threshold and returns the first day where the average temperature exceeds the threshold. If no such day exists, return -1.

EXAMPLES:
Input: 
```python
temperatures = [70, 75, 68, 85, 90, 88, 76]
threshold = 80
```
Output: 
```python
3
```
Explanation: The first day where the average temperature exceeds 80 is the 4th day (index 3, which has a temperature of 85).

Input: 
```python
temperatures = [65, 70, 72, 74, 76]
threshold = 78
```
Output: 
```python
-1
```
Explanation: There are no days where the average temperature exceeds 78.

CONSTRAINTS:
- 1 <= len(temperatures) <= 1000
- -100 <= temperatures[i] <= 100
- 0 <= threshold <= 100

HINT:
Try to think of how binary search can help you efficiently find the first index exceeding the threshold.

SOLUTION_PYTHON:
```python
def find_first_exceeding_day(temperatures, threshold):
    left, right = 0, len(temperatures) - 1
    result = -1
    
    while left <= right:
        mid = left + (right - left) // 2
        
        if temperatures[mid] > threshold:
            result = mid  # Possible candidate
            right = mid - 1  # Look for an earlier occurrence
        else:
            left = mid + 1  # Search in the right half

    return result
```

SOLUTION_JS:
```javascript
function findFirstExceedingDay(temperatures, threshold) {
    let left = 0;
    let right = temperatures.length - 1;
    let result = -1;

    while (left <= right) {
        const mid = Math.floor((left + right) / 2);

        if (temperatures[mid] > threshold) {
            result = mid; // Possible candidate
            right = mid - 1; // Look for an earlier occurrence
        } else {
            left = mid + 1; // Search in the right half
        }
    }

    return result;
}
```

TIME_COMPLEXITY: O(log n)
SPACE_COMPLEXITY: O(1)

EXPLANATION:
The approach utilizes binary search to efficiently locate the first index where the average temperature exceeds the given threshold. By progressively narrowing down the search range based on comparison with the threshold, we achieve a logarithmic time complexity instead of a linear scan through the temperatures, making it optimal for larger datasets.
