TITLE: Maximum Sum Path in a Binary Tree
DIFFICULTY: Hard
TOPIC: Trees & Graphs
ESTIMATED_TIME: 45 minutes

PROBLEM:
You are tasked with finding the maximum sum path in a binary tree, where the path can start at any node and can travel up to any node in the tree, moving only downwards (from parent to child). The path does not include any cycles and can consist of a single node or multiple connected nodes. Your goal is to return the maximum sum of the nodes along any such path.

You are given a binary tree node definition as follows:

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
```

EXAMPLES:
Input: 
```python
root = TreeNode(10)
root.left = TreeNode(2)
root.right = TreeNode(10)
root.left.left = TreeNode(20)
root.left.right = TreeNode(1)
root.right.right = TreeNode(-25)
root.right.right.left = TreeNode(3)
root.right.right.right = TreeNode(4)
```
Output: 42
Explanation: The path that results in the maximum sum is: 10 -> 2 -> 20 (10 + 2 + 20 = 32), or 10 -> 10 -> 4 (10 + 10 + 4 = 24). The maximum sum that can be achieved is 42 (10 + 10 + 4).

Input: 
```python
root = TreeNode(-2)
root.left = TreeNode(-1)
```
Output: -1
Explanation: The maximum sum from the only obtainable paths (which only includes negative values) is the single node -1.

CONSTRAINTS:
- The number of nodes in the tree is in the range [1, 1000].
- Each node's value is an integer within the range [-1000, 1000].

HINT:
Consider using a depth-first search to explore all possible paths and keep track of the maximum sum encountered.

SOLUTION_PYTHON:
```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def max_sum_path(root: TreeNode) -> int:
    """
    Function to find the maximum sum path in a binary tree.
    """
    global_max = float('-inf')

    def dfs(node: TreeNode) -> int:
        nonlocal global_max
        if not node:
            return 0
        
        # Recursive DFS to find maximum path sums in left and right subtrees
        left_max = max(dfs(node.left), 0)  # Only consider positive sums
        right_max = max(dfs(node.right), 0)

        # Update global maximum path sum considering paths through the current node
        global_max = max(global_max, node.val + left_max + right_max)

        # Return the maximum path sum considering the current node
        return node.val + max(left_max, right_max)

    dfs(root)
    return global_max
```

SOLUTION_JS:
```javascript
class TreeNode {
    constructor(val = 0, left = null, right = null) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

var maxSumPath = function(root) {
    let globalMax = -Infinity;

    function dfs(node) {
        if (!node) return 0;

        // Compute maximum path sums in left and right subtrees
        let leftMax = Math.max(dfs(node.left), 0);
        let rightMax = Math.max(dfs(node.right), 0);

        // Update the global maximum path sum
        globalMax = Math.max(globalMax, node.val + leftMax + rightMax);

        // Return the maximum path sum including the current node
        return node.val + Math.max(leftMax, rightMax);
    }

    dfs(root);
    return globalMax;
};
```

TIME_COMPLEXITY: O(n)  
SPACE_COMPLEXITY: O(h) where h is the height of the binary tree (due to call stack in recursion).

EXPLANATION:
The approach involves a depth-first search (DFS) traversal of the tree to explore all paths. For each node, the maximum sums from the left and right children are computed and added to the current node's value. This helps ensure that only the highest contribution from child paths is included. The global maximum is updated if the sum including the current node exceeds it. This effectively captures the highest path sums throughout the binary tree.
