TITLE: Minimal Cost Path with Obstacles
DIFFICULTY: Easy
TOPIC: Dynamic Programming
ESTIMATED_TIME: 30 minutes

PROBLEM:
You are given a 2D grid representing a map where each cell contains a non-negative integer representing the cost to enter that cell. You start from the top-left corner of the grid and can only move to the right or down. Additionally, there are certain cells marked as obstacles (represented by `-1`) that cannot be traversed. Your task is to find the minimum cost path to reach the bottom-right corner of the grid. If there is no valid path to the destination, return `-1`.

EXAMPLES:
Input: 
```
grid = [
    [1, 3, 1],
    [1, -1, 2],
    [4, 2, 1]
]
```
Output: `7`
Explanation: The minimum cost path is 1 -> 3 -> 2 -> 1.

Input: 
```
grid = [
    [1, 2, 3],
    [4, -1, 6],
    [7, 8, 9]
]
```
Output: `-1`
Explanation: There is no path to reach the bottom-right corner because of the obstacle.

CONSTRAINTS:
- The dimensions of the grid will be at most 100 x 100.
- The values in the grid will be between 0 and 100, with obstacles marked by `-1`.

HINT:
Consider using a dynamic programming table to keep track of the minimum costs to reach each cell from the start.

SOLUTION_PYTHON:
```python
def minCost(grid):
    if not grid or grid[0][0] == -1 or grid[-1][-1] == -1:
        return -1
    
    rows, cols = len(grid), len(grid[0])
    dp = [[float('inf')] * cols for _ in range(rows)]
    dp[0][0] = grid[0][0]
    
    for i in range(rows):
        for j in range(cols):
            if grid[i][j] == -1:
                continue
            if i > 0:
                dp[i][j] = min(dp[i][j], dp[i - 1][j] + grid[i][j])
            if j > 0:
                dp[i][j] = min(dp[i][j], dp[i][j - 1] + grid[i][j])
    
    return dp[-1][-1] if dp[-1][-1] != float('inf') else -1

# Time Complexity: O(m*n)
# Space Complexity: O(m*n)
```

SOLUTION_JS:
```javascript
function minCost(grid) {
    if (!grid || grid[0][0] === -1 || grid[grid.length - 1][grid[0].length - 1] === -1) {
        return -1;
    }
    
    const rows = grid.length;
    const cols = grid[0].length;
    const dp = Array.from({ length: rows }, () => Array(cols).fill(Infinity));
    dp[0][0] = grid[0][0];
    
    for (let i = 0; i < rows; i++) {
        for (let j = 0; j < cols; j++) {
            if (grid[i][j] === -1) {
                continue;
            }
            if (i > 0) {
                dp[i][j] = Math.min(dp[i][j], dp[i - 1][j] + grid[i][j]);
            }
            if (j > 0) {
                dp[i][j] = Math.min(dp[i][j], dp[i][j - 1] + grid[i][j]);
            }
        }
    }
    
    return dp[rows - 1][cols - 1] !== Infinity ? dp[rows - 1][cols - 1] : -1;
}

// Time Complexity: O(m*n)
// Space Complexity: O(m*n)
```

TIME_COMPLEXITY: O(m*n)
SPACE_COMPLEXITY: O(m*n)

EXPLANATION:
The algorithm uses a dynamic programming approach to fill a 2D table where each entry represents the minimum cost to reach that cell from the top-left corner. It checks the costs from the cell above and the cell to the left, updating the current cell's cost if a cheaper path is found. The final result is derived from the bottom-right corner of the table. This approach efficiently determines the minimum cost while appropriately handling obstacles.
