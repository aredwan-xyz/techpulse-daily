TITLE: Longest Substring with K Distinct Characters
DIFFICULTY: Hard
TOPIC: Sliding Window
ESTIMATED_TIME: 35 minutes

PROBLEM:
You are tasked with processing a stream of character data that comes in as a string. Your goal is to find the longest substring that contains at most K distinct characters. This task is crucial for developing features such as character frequency analysis in text processing. You need to return the longest substring's length that satisfies the condition.

EXAMPLES:
Input: "abcabcbb", K = 2  
Output: 4  
Explanation: The longest substring with at most 2 distinct characters is "abab", which has a length of 4.

Input: "aabbcc", K = 3  
Output: 6  
Explanation: The entire string is valid since it contains 3 distinct characters, thus its length is 6.

CONSTRAINTS:
- The input string will have a length between 1 and 10^5.
- K will be a positive integer less than or equal to the length of the string.

HINT:
Consider maintaining a frequency count of the characters in the current window and adjust your window size based on the number of distinct characters.

SOLUTION_PYTHON:
```python
def longest_substring_k_distinct(s: str, K: int) -> int:
    left, right = 0, 0
    char_count = {}
    max_length = 0

    while right < len(s):
        char_count[s[right]] = char_count.get(s[right], 0) + 1

        while len(char_count) > K:
            char_count[s[left]] -= 1
            if char_count[s[left]] == 0:
                del char_count[s[left]]
            left += 1
        
        max_length = max(max_length, right - left + 1)
        right += 1

    return max_length

# Example usage:
# print(longest_substring_k_distinct("abcabcbb", 2))  # Output: 4
```

SOLUTION_JS:
```javascript
function longestSubstringKDistinct(s, K) {
    let left = 0, right = 0;
    const charCount = {};
    let maxLength = 0;

    while (right < s.length) {
        charCount[s[right]] = (charCount[s[right]] || 0) + 1;

        while (Object.keys(charCount).length > K) {
            charCount[s[left]]--;
            if (charCount[s[left]] === 0) {
                delete charCount[s[left]];
            }
            left++;
        }

        maxLength = Math.max(maxLength, right - left + 1);
        right++;
    }

    return maxLength;
}

// Example usage:
// console.log(longestSubstringKDistinct("abcabcbb", 2));  // Output: 4
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(K)

EXPLANATION:
The algorithm utilizes the sliding window technique by maintaining two pointers, left and right, that define the current substring. As the right pointer expands the window by moving forward, the algorithm checks if the number of distinct characters exceeds K. If it does, the left pointer is incremented until the condition is satisfied again. This results in an optimal time complexity of O(n) as each character in the string is processed a constant number of times.
