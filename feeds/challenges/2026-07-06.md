TITLE: Remove Duplicates in a Linked List and Maintain Unique Order  
DIFFICULTY: Easy  
TOPIC: Linked Lists  
ESTIMATED_TIME: 30 minutes  

PROBLEM:  
You are given a singly linked list that may contain duplicate values. Your task is to create a new linked list that contains only the unique values from the original list while maintaining the order in which they first appear. The duplicate values should be omitted from the new list altogether. 

Implement a function `remove_duplicates(head)` that takes the head of the linked list as input and returns the head of a new linked list with duplicates removed.

EXAMPLES:  
Input: `[1 -> 2 -> 3 -> 2 -> 1]`  
Output: `[1 -> 2 -> 3]`  
Explanation: The duplicates `1` and `2` (the second occurrences) are removed, and the order of appearance is preserved.

Input: `[4 -> 5 -> 5 -> 6 -> 4 -> 7]`  
Output: `[4 -> 5 -> 6 -> 7]`  
Explanation: The duplicates `5` and `4` (the second occurrences) are removed, with the unique values kept in their original order.

CONSTRAINTS:  
- The linked list can contain at most 1000 nodes.  
- The values of the nodes will be integers in the range `0` to `1000`.  

HINT:  
Consider using a set to keep track of the values that have already been seen in order to filter out duplicates efficiently.

SOLUTION_PYTHON:
```python
class ListNode:
    def __init__(self, value=0, next=None):
        self.value = value
        self.next = next

def remove_duplicates(head: ListNode) -> ListNode:
    seen = set()  # To keep track of seen values
    dummy = ListNode(0)  # Create a dummy node to simplify list construction
    current = dummy  # Pointer for construction of the new list
    
    while head is not None:
        if head.value not in seen:
            seen.add(head.value)  # Add to seen values
            current.next = ListNode(head.value)  # Add unique value to new list
            current = current.next  # Move forward in the new list
        head = head.next  # Move to the next node in the original list
    
    return dummy.next  # Return head of the new list
```

SOLUTION_JS:
```javascript
class ListNode {
    constructor(value = 0, next = null) {
        this.value = value;
        this.next = next;
    }
}

function removeDuplicates(head) {
    const seen = new Set();  // To keep track of seen values
    const dummy = new ListNode(0);  // Create a dummy node to simplify list construction
    let current = dummy;  // Pointer for construction of the new list
    
    while (head) {
        if (!seen.has(head.value)) {
            seen.add(head.value);  // Add to seen values
            current.next = new ListNode(head.value);  // Add unique value to new list
            current = current.next;  // Move forward in the new list
        }
        head = head.next;  // Move to the next node in the original list
    }
    
    return dummy.next;  // Return head of the new list
}
```

TIME_COMPLEXITY: O(n)  
SPACE_COMPLEXITY: O(n)  

EXPLANATION:  
The solution uses a single pass through the linked list, achieving O(n) time complexity where n is the number of nodes in the list. A set is utilized to record values that have been observed, ensuring duplicates are filtered out effectively. The dummy head simplifies the list construction, allowing easy handling of the new linked list without needing to check for edge cases related to the head node.

FOLLOW_UP:
