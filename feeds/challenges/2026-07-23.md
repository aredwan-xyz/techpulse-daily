TITLE: Generate Palindromic Combinations  
DIFFICULTY: Medium  
TOPIC: Recursion & Backtracking  
ESTIMATED_TIME: 30 minutes  

PROBLEM:  
Given a string containing lowercase letters, your task is to generate all possible unique palindromic combinations that can be formed by rearranging the letters of the string. A palindromic combination is a string that reads the same forwards and backwards. Return the combinations as a list of strings. The output should be sorted in lexicographical order.

EXAMPLES:  
Input: "aabb"  
Output: ["abba", "baab"]  
Explanation: The possible unique palindromic combinations that can be formed by rearranging "aabb" are "abba" and "baab".

Input: "abc"  
Output: []  
Explanation: There are no possible palindromic combinations since the characters cannot form a palindrome.

CONSTRAINTS:  
- The input string will contain at most 15 lowercase letters.  
- Use a maximum of O(n) additional space for your solution.  

HINT:  
Consider the characteristics of palindromes while determining which characters can be at the center.

SOLUTION_PYTHON:
```python
def generate_palindromic_combinations(s):
    from collections import Counter
    result = []
    
    # Helper function for backtracking
    def backtrack(path, counter, length):
        if len(path) == length:
            result.append(''.join(path))
            return
        
        for char in counter:
            if counter[char] > 0:
                # Use the character
                path.append(char)
                counter[char] -= 1
                
                # Continue backtracking
                backtrack(path, counter, length)
                
                # Backtrack
                path.pop()
                counter[char] += 1
    
    # Count occurrences of each character
    char_count = Counter(s)
    odd_count = sum(1 for count in char_count.values() if count % 2 != 0)
    
    # If more than one character has an odd count, no palindrome can be formed
    if odd_count > 1:
        return []

    # Prepare the half string and handle odd-length string
    half_path = []
    half_length = sum(count // 2 for count in char_count.values())
    
    # Start backtracking
    backtrack(half_path, char_count, half_length)
    
    # Create full palindromic combinations
    full_result = []
    for half in result:
        # Create the palindrome from the half
        full_result.append(half + half[::-1])  # Reflect the first half

    return sorted(full_result)

# Example Usage
print(generate_palindromic_combinations("aabb"))  # Output: ['abba', 'baab']
```

SOLUTION_JS:
```javascript
function generatePalindromicCombinations(s) {
    const result = [];
    const charCount = {};
    
    // Count occurrences of each character
    for (const char of s) {
        charCount[char] = (charCount[char] || 0) + 1;
    }
    
    const oddCount = Object.values(charCount).filter(count => count % 2 !== 0).length;

    // If more than one character has an odd count, no palindrome can be formed
    if (oddCount > 1) {
        return [];
    }

    function backtrack(path, counter, length) {
        if (path.length === length) {
            result.push(path.join(''));
            return;
        }
        
        for (const char in counter) {
            if (counter[char] > 0) {
                // Use the character
                path.push(char);
                counter[char] -= 1;
                
                // Continue backtracking
                backtrack(path, counter, length);
                
                // Backtrack
                path.pop();
                counter[char] += 1;
            }
        }
    }

    // Preparing for backtracking
    const halfPath = [];
    const halfLength = Object.values(charCount).reduce((acc, count) => acc + Math.floor(count / 2), 0);
    
    backtrack(halfPath, charCount, halfLength);

    // Create full palindromic combinations
    const fullResult = [];
    for (const half of result) {
        fullResult.push(half + half.split('').reverse().join('')); // Reflect the first half
    }

    return fullResult.sort();
}

// Example Usage
console.log(generatePalindromicCombinations("aabb"));  // Output: ['abba', 'baab']
```

TIME_COMPLEXITY: O(n!)  
SPACE_COMPLEXITY: O(n)  

EXPLANATION:  
The approach uses recursion and backtracking to explore all possible arrangements of characters that can form palindromic combinations. Due to the nature of palindromes, we can create a half-string using character counts, and reflect it to generate the full palindromic combination. Since the maximum possible unique arrangements is factorial in nature, the algorithm has a time complexity of O(n!), but generating the half-string reduces computations. The space complexity is linear relative to the number of unique characters used during the arrangements.
