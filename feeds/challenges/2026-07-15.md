TITLE: Unique Character Counter
DIFFICULTY: Easy
TOPIC: Arrays & Strings
ESTIMATED_TIME: 15 minutes

PROBLEM:
You are given a string input. Your task is to count the number of unique characters in the string while ignoring spaces and punctuation. Return the count of these unique characters. For this problem, a character is considered unique if it appears exactly once in the string (case-sensitive).

EXAMPLES:
Input: "Hello, World!"
Output: 7
Explanation: The unique characters are 'H', 'e', 'o', 'W', 'r', 'l', and 'd'. 

Input: "Programming!"
Output: 3
Explanation: The unique characters are 'P', 'r', and 'g' (case-sensitive and only 'g' appears once). 

CONSTRAINTS:
- The string length will be between 1 and 1000 characters.
- The characters in the string can include alphabetic characters, digits, spaces, and punctuation.

HINT:
Consider using a dictionary or a counter to help keep track of the character frequencies.

SOLUTION_PYTHON:
```python
def unique_character_count(s: str) -> int:
    from collections import Counter
    
    # Remove spaces and punctuation, convert to lower case
    clean_string = ''.join(filter(str.isalnum, s))
    
    # Count the frequency of each character
    char_count = Counter(clean_string)
    
    # Count characters that appear exactly once
    unique_count = sum(1 for count in char_count.values() if count == 1)
    
    return unique_count
```

SOLUTION_JS:
```javascript
function uniqueCharacterCount(s) {
    const charCount = {};
    
    // Clean the string and count character frequencies
    for (let char of s) {
        if (char.match(/[a-zA-Z0-9]/)) { // Only count alphanumeric characters
            charCount[char] = (charCount[char] || 0) + 1;
        }
    }
    
    // Count characters that appear exactly once
    let uniqueCount = 0;
    for (let count of Object.values(charCount)) {
        if (count === 1) {
            uniqueCount++;
        }
    }
    
    return uniqueCount;
}
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The approach uses a dictionary to count the occurrences of each alphanumeric character while ignoring spaces and punctuation. After counting, we simply iterate through the frequency dictionary to tally the number of characters that appear exactly once. This method ensures we only traverse the string a couple of times, making it efficient.
