TITLE: Time Series Anomaly Detection
DIFFICULTY: Hard

PROBLEM:
You are given a time series dataset representing daily stock prices of a company over a period of time. Your task is to implement an anomaly detection algorithm that identifies significant deviations from the expected trend. You need to compute the rolling mean and rolling standard deviation over a specified window size. Any data point that falls outside 2 standard deviations from the rolling mean is considered an anomaly.

Write a function that takes in the time series data and the window size and returns a list of indices where anomalies occur.

EXAMPLES:
Input: 
```python
data = [100, 102, 101, 103, 200, 102, 101, 99, 98, 100]
window_size = 3
```
Output: 
```python
[4]
```
Explanation: The value 200 at index 4 is more than 2 standard deviations away from the rolling mean of the previous window [100, 102, 101]. 

Input: 
```python
data = [50, 51, 52, 53, 54, 50, 200, 52, 51]
window_size = 4
```
Output: 
```python
[6]
```
Explanation: The value 200 at index 6 is an anomaly as it is significantly greater than the rolling mean of the prior indices.

CONSTRAINTS:
- The length of the input data will be between 10 to 10^5.
- The window size will be a positive integer smaller than the length of the data.

HINT:
Consider using Pandas' rolling functionality to help compute the rolling mean and standard deviation efficiently.

SOLUTION_PYTHON:
```python
import pandas as pd

def detect_anomalies(data, window_size):
    # Convert the list into a Pandas DataFrame for easier manipulation
    df = pd.DataFrame(data, columns=['Price'])
    
    # Calculate the rolling mean and rolling standard deviation
    df['RollingMean'] = df['Price'].rolling(window=window_size).mean()
    df['RollingStd'] = df['Price'].rolling(window=window_size).std()
    
    # Detect anomalies by comparing the data points with the rolling mean & stddev
    anomalies = df[(df['Price'] > df['RollingMean'] + 2 * df['RollingStd']) | 
                   (df['Price'] < df['RollingMean'] - 2 * df['RollingStd'])]
    
    # Return the indices of the anomalies
    return anomalies.index.tolist()

# Time Complexity: O(n) where n is the number of data points
# Space Complexity: O(n) for storing the DataFrame
```

SOLUTION_JS:
```javascript
function detectAnomalies(data, windowSize) {
    let anomalies = [];
    
    for (let i = windowSize; i < data.length; i++) {
        // Get the current window
        const window = data.slice(i - windowSize, i);
        
        // Calculate mean and standard deviation for the window
        const mean = window.reduce((a, b) => a + b, 0) / windowSize;
        const stdDev = Math.sqrt(window.reduce((sq, n) => sq + (n - mean) ** 2, 0) / windowSize);
        
        // Check if the current point is an anomaly
        if (data[i] > (mean + 2 * stdDev) || data[i] < (mean - 2 * stdDev)) {
            anomalies.push(i);
        }
    }
    
    return anomalies;
}

// Time Complexity: O(n * m) where n is the total number of data points and m is the window size
// Space Complexity: O(1) - we only return indices
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The proposed approach efficiently calculates the rolling mean and rolling standard deviation using a sliding window technique, allowing for quick assessment of anomalies in the dataset. By traversing the array once, while maintaining the necessary statistics, it achieves linear time complexity which is optimal for large datasets.
