TITLE: Inventory Reordering
DIFFICULTY: Medium
TOPIC: Sorting & Searching
ESTIMATED_TIME: 30 minutes

PROBLEM:
You are managing an inventory system for a retail store, and you need to help optimize the ordering of stock before significant sales events like Black Friday. Given an array of items with their current stock levels and a threshold, your task is to reorder items that fall below this threshold. You want to return a sorted list of items that need to be reordered based on their current stock levels. If two items have the same stock level, they should appear in alphabetical order.

Write a function `reorder_items(items: List[str], stock: List[int], threshold: int) -> List[str]` that takes in the list of item names and their respective stock levels, and the threshold for reordering.

EXAMPLES:
Input: `items = ["apple", "banana", "orange", "kiwi"]`, `stock = [10, 2, 5, 2]`, `threshold = 5`
Output: `["banana", "kiwi", "orange"]`
Explanation: "banana" (2), "kiwi" (2), and "orange" (5) are below the threshold of 5. In sorted order: ["banana", "kiwi", "orange"].

Input: `items = ["grapes", "peach", "mango", "blueberry"]`, `stock = [0, 2, 1, 3]`, `threshold = 2`
Output: `["blueberry", "grapes", "mango"]`
Explanation: "grapes" (0) and "mango" (1) are below the threshold of 2. In alphabetical order: ["blueberry", "grapes", "mango"].

CONSTRAINTS:
- `1 ≤ len(items) ≤ 10^5`
- Each item name consists of only lowercase English letters and has a length of at most 100.

HINT:
Consider using a combination of filtering and sorting to achieve your goal efficiently.

SOLUTION_PYTHON:
```python
def reorder_items(items: list[str], stock: list[int], threshold: int) -> list[str]:
    # Create a list of items that need to be reordered
    reorder_list = [item for item, qty in zip(items, stock) if qty < threshold]
    # Sort the list alphabetically
    reorder_list.sort()
    return reorder_list
```

SOLUTION_JS:
```javascript
function reorderItems(items, stock, threshold) {
    // Filter items below the threshold and sort them
    return items
        .filter((item, index) => stock[index] < threshold)
        .sort();
}
```

TIME_COMPLEXITY: O(n log n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The approach involves first filtering out the items whose stock level is below the specified threshold. This can be done efficiently in a single pass over the data. After filtering, the resulting list of items is sorted alphabetically. The overall time complexity is dominated by the sorting step, resulting in O(n log n), which is efficient given the constraints.
