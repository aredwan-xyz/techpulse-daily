PROBLEM:  
You are working for a retail company that wants to understand how weather conditions affect sales. You have been provided with two datasets: one that contains daily sales data over the last two years and another containing daily weather data including temperature, precipitation, and wind speed for the same period. Your task is to analyze these datasets and generate insights.

You need to create a function that calculates the correlation between daily weather features (temperature, precipitation, and wind speed) and total sales for each day. The function should return a summary DataFrame showing the average sales, average temperature, average precipitation, and average wind speed for each weather metric, along with their correlation coefficients.

HINT:  
Consider using the `merge` function from Pandas to combine the datasets before calculating the correlations.

SOLUTION_PYTHON:
```python
import pandas as pd

def analyze_weather_sales(sales_data: pd.DataFrame, weather_data: pd.DataFrame) -> pd.DataFrame:
    # Convert date column to datetime type
    sales_data['date'] = pd.to_datetime(sales_data['date'])
    weather_data['date'] = pd.to_datetime(weather_data['date'])
    
    # Merge sales and weather data on the date
    combined_data = pd.merge(sales_data.groupby('date').sum().reset_index(), weather_data, on='date')
    
    # Calculate averages for each weather feature
    avg_sales = combined_data['sales'].mean()
    summary = []
    
    for weather_metric in ['temperature', 'precipitation', 'wind_speed']:
        avg_weather_value = combined_data[weather_metric].mean()
        correlation = combined_data['sales'].corr(combined_data[weather_metric])
        summary.append({
            'weather_metric': weather_metric,
            'avg_sales': avg_sales,
            'avg_temperature': avg_weather_value if weather_metric == 'temperature' else avg_sales,
            'avg_precipitation': avg_weather_value if weather_metric == 'precipitation' else avg_sales,
            'avg_wind_speed': avg_weather_value if weather_metric == 'wind_speed' else avg_sales,
            'correlation': correlation
        })
    
    # Create and return summary DataFrame
    return pd.DataFrame(summary)

# Note: The above function can be directly used with sales_data and weather_data DataFrames.
```
