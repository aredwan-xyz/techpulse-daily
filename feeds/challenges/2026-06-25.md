TITLE: Backtrack to Balance
DIFFICULTY: Medium
TOPIC: Recursion & Backtracking
ESTIMATED_TIME: 30 minutes

PROBLEM:
You have a list of integers that represent balance changes in a financial account over a series of months. Your objective is to find all unique subsets of these integers such that the sum of the integers in each subset equals zero. This is essential to identify potential transactions that could lead to a neutral account status over time. Your program should return the unique subsets in ascending order.

EXAMPLES:
Input: [2, -2, 1, -1, 1]
Output: [[-2, 2], [-1, 1]]
Explanation: The subsets [-2, 2] and [-1, 1] both sum up to zero. Note that subsets should be unique.

Input: [3, 0, -3]
Output: [[-3, 3], [0]]
Explanation: The subsets [-3, 3] and [0] both sum to zero. The order of subsets and elements within them must be unique.

CONSTRAINTS:
- The input list will have a length between 1 and 20.
- All integers in the list will be in the range of -1000 to 1000.

HINT:
Consider using a recursive approach to explore each integer by either including it in the current subset or skipping it.

SOLUTION_PYTHON:
```python
def backtrack_to_balance(nums):
    def backtrack(start, current_sum, path):
        if current_sum == 0 and path:
            result.append(sorted(path))
            return
        for i in range(start, len(nums)):
            # To prevent duplicates, we skip the same number
            if i > start and nums[i] == nums[i-1]:
                continue
            backtrack(i + 1, current_sum + nums[i], path + [nums[i]])
    
    nums.sort()  # Sort the input to facilitate duplicate handling
    result = []
    backtrack(0, 0, [])
    # Use set to remove duplicates, then convert back to list of lists
    result_set = list(map(list, set(tuple(sorted(r)) for r in result)))
    return sorted(result_set)

# Example Usage
print(backtrack_to_balance([2, -2, 1, -1, 1]))  # Output: [[-2, 2], [-1, 1]]
```

SOLUTION_JS:
```javascript
function backtrackToBalance(nums) {
    const result = [];
    const backtrack = (start, currentSum, path) => {
        if (currentSum === 0 && path.length) {
            result.push([...path]);
            return;
        }
        for (let i = start; i < nums.length; i++) {
            // Skip duplicates
            if (i > start && nums[i] === nums[i - 1]) continue;
            path.push(nums[i]);
            backtrack(i + 1, currentSum + nums[i], path);
            path.pop();
        }
    };

    nums.sort((a, b) => a - b);
    backtrack(0, 0, []);
    // Remove duplicates from result
    const uniqueResults = Array.from(new Set(result.map(r => JSON.stringify(r)))).map(e => JSON.parse(e));
    return uniqueResults.sort((a, b) => a[0] - b[0]);
}

// Example Usage
console.log(backtrackToBalance([2, -2, 1, -1, 1]));  // Output: [[-2, 2], [-1, 1]]
```

TIME_COMPLEXITY: O(2^N)
SPACE_COMPLEXITY: O(N)

EXPLANATION:
The solution employs backtracking to explore all possible subsets of the integer list. Each recursive call either includes or excludes a number from the current subset. By sorting the list, duplicate subsets are avoided. The results are filtered for uniqueness, ensuring that all returned subsets have distinct elements and sums.
