TITLE: Sorted Squares
DIFFICULTY: Medium
TOPIC: Two Pointers
ESTIMATED_TIME: 30 minutes

PROBLEM:
You are given a sorted list of integers `nums`, which can contain negative numbers. Your task is to return a new list containing the squares of each number, sorted in non-decreasing order. You must implement this using the two-pointer technique to achieve optimal time complexity.

EXAMPLES:
Input: nums = [-4, -1, 0, 3, 10]
Output: [0, 1, 9, 16, 100]
Explanation: Squaring each number gives [16, 1, 0, 9, 100], and sorting them results in [0, 1, 9, 16, 100].

Input: nums = [-7, -3, 2, 3, 11]
Output: [4, 9, 9, 49, 121]
Explanation: Squaring each number results in [49, 9, 4, 9, 121], and sorting yields [4, 9, 9, 49, 121].

CONSTRAINTS:
- The length of nums will be in the range [1, 10^5].
- Each integer in nums will be in the range [-10^4, 10^4].

HINT:
Consider using two pointers starting from the beginning and end of the list to process the squares in order.

SOLUTION_PYTHON:
```python
def sorted_squares(nums):
    n = len(nums)
    res = [0] * n
    left, right = 0, n - 1
    position = n - 1

    while left <= right:
        left_square = nums[left] ** 2
        right_square = nums[right] ** 2
        
        if left_square > right_square:
            res[position] = left_square
            left += 1
        else:
            res[position] = right_square
            right -= 1
        position -= 1

    return res

# Time complexity: O(n)
# Space complexity: O(n)
```

SOLUTION_JS:
```javascript
function sortedSquares(nums) {
    const n = nums.length;
    const res = new Array(n);
    let left = 0, right = n - 1, position = n - 1;

    while (left <= right) {
        const leftSquare = nums[left] ** 2;
        const rightSquare = nums[right] ** 2;

        if (leftSquare > rightSquare) {
            res[position] = leftSquare;
            left++;
        } else {
            res[position] = rightSquare;
            right--;
        }
        position--;
    }

    return res;
}

// Time complexity: O(n)
// Space complexity: O(n)
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The solution employs a two-pointer strategy where one pointer starts from the beginning of the array (`left`) and the other from the end (`right`). As we compare the squares of the two numbers, we place the larger square at the end of the result list. This method efficiently fills the result from the back to the front, ensuring it remains sorted without needing to sort at the end.

FOLLOW_UP:
