TITLE: Balanced City Planning
DIFFICULTY: Hard
TOPIC: Trees & Graphs
ESTIMATED_TIME: 60 minutes

PROBLEM:
You are a city planner responsible for ensuring that all districts in a city are balanced in terms of resources such as parks, schools, and hospitals. Each district is represented as a node in a tree structure, with connections indicating direct access between districts. The city wants to maintain a balanced allocation of resources such that the difference in the number of resources between any two connected districts does not exceed a specified limit, `maxDiff`.

Your goal is to determine if it's possible to allocate resources in such a way that the balance condition holds for all pairs of directly connected districts. You need to return `True` if it’s possible to allocate resources within the constraints, and `False` otherwise.

EXAMPLES:
Input: 
```python
n = 5
edges = [(0, 1), (0, 2), (1, 3), (1, 4)]
resources = [10, 15, 20, 15, 10]
maxDiff = 5
```
Output: 
```python
True
```
Explanation: The tree is formed by the edges between districts. For each pair of connected districts, the resource differences are within the limit of `5`.

Input: 
```python
n = 4
edges = [(0, 1), (1, 2), (1, 3)]
resources = [10, 15, 25, 10]
maxDiff = 5
```
Output: 
```python
False
```
Explanation: The connection between district `1` (15 resources) and district `2` (25 resources) has a difference of `10`, which exceeds the maximum allowed difference of `5`.

CONSTRAINTS:
- The number of districts `n` (1 <= n <= 1000)
- The resources list will have `n` integers, each ranging between `0` and `10^9`
- `maxDiff` is a non-negative integer

HINT:
Consider performing a traversal of the tree while checking the resource balance constraints between connected districts.

SOLUTION_PYTHON:
```python
def can_balance_resources(n, edges, resources, maxDiff):
    from collections import defaultdict, deque

    # Construct the adjacency list for the tree
    graph = defaultdict(list)
    
    for u, v in edges:
        graph[u].append(v)
        graph[v].append(u)

    visited = [False] * n
    
    # BFS or DFS to traverse the tree
    def bfs(start):
        queue = deque([start])
        visited[start] = True
        
        while queue:
            node = queue.popleft()
            current_resource = resources[node]

            for neighbor in graph[node]:
                if not visited[neighbor]:   
                    # Check resource difference
                    if abs(current_resource - resources[neighbor]) > maxDiff:
                        return False
                    visited[neighbor] = True
                    queue.append(neighbor)

        return True

    # Start BFS/DFS from the first node
    return bfs(0)

# Example usage
print(can_balance_resources(5, [(0, 1), (0, 2), (1, 3), (1, 4)], [10, 15, 20, 15, 10], 5))  # True
print(can_balance_resources(4, [(0, 1), (1, 2), (1, 3)], [10, 15, 25, 10], 5))  # False
```

SOLUTION_JS:
```javascript
function canBalanceResources(n, edges, resources, maxDiff) {
    const graph = Array.from({ length: n }, () => []);
    for (const [u, v] of edges) {
        graph[u].push(v);
        graph[v].push(u);
    }

    const visited = new Array(n).fill(false);

    function bfs(start) {
        const queue = [start];
        visited[start] = true;

        while (queue.length > 0) {
            const node = queue.shift();
            const currentResource = resources[node];

            for (const neighbor of graph[node]) {
                if (!visited[neighbor]) {
                    if (Math.abs(currentResource - resources[neighbor]) > maxDiff) {
                        return false;
                    }
                    visited[neighbor] = true;
                    queue.push(neighbor);
                }
            }
        }

        return true;
    }

    // Start BFS from the first node
    return bfs(0);
}

// Example usage
console.log(canBalanceResources(5, [[0, 1], [0, 2], [1, 3], [1, 4]], [10, 15, 20, 15, 10], 5)); // true
console.log(canBalanceResources(4, [[0, 1], [1, 2], [1, 3]], [10, 15, 25, 10], 5)); // false
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The problem can be efficiently solved using a breadth-first search (BFS) or depth-first search (DFS) approach to traverse the tree structure. For each connected district, we check if the difference in resources remains within the allowable limit (`maxDiff`). Since we only visit each node and edge once, the solution runs in linear time relative to the number of districts.
