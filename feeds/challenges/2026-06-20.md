TITLE: Maximum Distinct Fruit in Baskets  
DIFFICULTY: Hard  
TOPIC: Sliding Window  
ESTIMATED_TIME: 30 minutes  

PROBLEM:  
You are given an array of integers representing different types of fruits in a garden, where each integer corresponds to a type of fruit. Your task is to determine the maximum number of distinct fruit types you can collect when you are allowed to gather fruit from at most two baskets at a time. The baskets can hold an unlimited number of fruits of the same type. You can only collect fruits if the types from the two baskets do not exceed more than two distinct types at any time.

Write a function that returns the maximum number of distinct fruit types that can be collected.

EXAMPLES:  
Input: `fruits = [1, 2, 1, 2, 3]`  
Output: `2`  
Explanation: You can collect fruits of type 1 and type 2 from the first four elements, resulting in 2 distinct types.

Input: `fruits = [1, 2, 3, 2, 2]`  
Output: `2`  
Explanation: You can collect types 2 and 3 together for 2 distinct types within the last four elements.

CONSTRAINTS:  
- The length of the fruits array will be between 1 and 10^5.  
- The types of fruits are represented by integers between 1 and 1000.  

HINT:  
Consider using a hashmap to keep track of the counts of each fruit type while maintaining the conditions for the two-basket limit.

SOLUTION_PYTHON:
```python
def max_distinct_fruits(fruits):
    left, right = 0, 0
    fruit_count = {}
    max_distinct = 0
    
    while right < len(fruits):
        # Add fruit to the basket
        if fruits[right] in fruit_count:
            fruit_count[fruits[right]] += 1
        else:
            fruit_count[fruits[right]] = 1
        
        # If there are more than 2 types of fruits, slide the left pointer
        while len(fruit_count) > 2:
            fruit_count[fruits[left]] -= 1
            if fruit_count[fruits[left]] == 0:
                del fruit_count[fruits[left]]
            left += 1
        
        max_distinct = max(max_distinct, len(fruit_count))
        right += 1
    
    return max_distinct

# Time complexity: O(n)
# Space complexity: O(1) - since there are at most 2 fruit types in the hashmap
```

SOLUTION_JS:
```javascript
function maxDistinctFruits(fruits) {
    let left = 0;
    let right = 0;
    const fruitCount = {};
    let maxDistinct = 0;

    while (right < fruits.length) {
        // Add fruit to the basket
        fruitCount[fruits[right]] = (fruitCount[fruits[right]] || 0) + 1;

        // If there are more than 2 types of fruits, slide the left pointer
        while (Object.keys(fruitCount).length > 2) {
            fruitCount[fruits[left]]--;
            if (fruitCount[fruits[left]] === 0) {
                delete fruitCount[fruits[left]];
            }
            left++;
        }

        maxDistinct = Math.max(maxDistinct, Object.keys(fruitCount).length);
        right++;
    }

    return maxDistinct;
}

// Time complexity: O(n)
// Space complexity: O(1) - since there are at most 2 fruit types in the hashmap
```

TIME_COMPLEXITY: O(n)  
SPACE_COMPLEXITY: O(1)  

EXPLANATION:  
The solution leverages a sliding window approach with a hashmap to maintain the count of each fruit type within the constraints of having at most two distinct types. The right pointer expands the window by incrementing, while the left pointer contracts it to ensure that the number of distinct types does not exceed two. This implementation efficiently traverses the list while keeping track of the maximum distinct types found at any point, resulting in optimal performance.
