TITLE: Array Partitioning for Balanced Customer Allocation
DIFFICULTY: Easy
TOPIC: Dynamic Programming
ESTIMATED_TIME: 30 minutes

PROBLEM:
A retail company wants to allocate its products across different customer segments in a way that minimizes the difference between the total weight of products allocated to the most loaded segment and the least loaded segment. Each customer in a segment has a certain demand represented by weights in an integer array. Your job is to partition the array into two segments such that the absolute difference between the total weights of these two segments is minimized.

Write a function `min_difference_partition(weights: List[int]) -> int` that returns the minimum absolute difference between the two segments' totals.

EXAMPLES:
Input: [1, 6, 11, 5]
Output: 1
Explanation: The array can be partitioned into [1, 5] and [6, 11], which gives a difference of |(1 + 5) - (6 + 11)| = |6 - 17| = 11.

Input: [2, 3, 4, 5]
Output: 0
Explanation: The array can be perfectly partitioned into [2, 3] and [4, 5], which gives a difference of |(2 + 3) - (4 + 5)| = |5 - 9| = 4.

CONSTRAINTS:
- The number of weights, n, is at least 1 and at most 100.
- The weights are integers in the range of 1 to 1000.

HINT:
Consider that the problem can be viewed as a subset sum problem where you attempt to find a subset of weights that closely matches half of the total weight.

SOLUTION_PYTHON:
```python
def min_difference_partition(weights):
    total_weight = sum(weights)
    n = len(weights)
    
    # Create a DP array to keep track of possible sums
    half_weight = total_weight // 2
    dp = [False] * (half_weight + 1)
    dp[0] = True
    
    for weight in weights:
        for j in range(half_weight, weight - 1, -1):
            dp[j] = dp[j] or dp[j - weight]

    # Find the largest j such that dp[j] is True
    for j in range(half_weight, -1, -1):
        if dp[j]:
            return total_weight - 2 * j

# Time complexity: O(n * total_weight / 2)
# Space complexity: O(total_weight / 2)
```

SOLUTION_JS:
```javascript
function minDifferencePartition(weights) {
    const totalWeight = weights.reduce((acc, weight) => acc + weight, 0);
    const halfWeight = Math.floor(totalWeight / 2);
    const dp = new Array(halfWeight + 1).fill(false);
    dp[0] = true;

    for (const weight of weights) {
        for (let j = halfWeight; j >= weight; j--) {
            dp[j] = dp[j] || dp[j - weight];
        }
    }

    for (let j = halfWeight; j >= 0; j--) {
        if (dp[j]) {
            return totalWeight - 2 * j;
        }
    }
    
    return totalWeight; // fallback case
}

// Time complexity: O(n * total_weight / 2)
// Space complexity: O(total_weight / 2)
```

TIME_COMPLEXITY: O(n * total_weight / 2)
SPACE_COMPLEXITY: O(total_weight / 2)

EXPLANATION:
The algorithm uses dynamic programming to find a subset of weights that approaches half of the total weight of the array. It maintains a boolean array indicating possible sums. The approach is optimal as it systematically checks all combinations of weights, ensuring minimal absolute difference can be found using the constraints provided.
