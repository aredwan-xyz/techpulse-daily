TITLE: Stack of Plates
DIFFICULTY: Hard
TOPIC: Stack & Queue
ESTIMATED_TIME: 45 minutes

PROBLEM:
You are designing a service that needs to manage a stack of plates in a restaurant. The restaurant has a fixed number of plates, and they can be stacked or unstacked. When plates are added, they must be stacked on top of each other, and when they are removed, they must be unstacked from the top. However, this service also requires that we keep track of the maximum capacity allowed for each stack. 

Implement a `PlateStack` class with the following operations:
- `stack(plates: List[int])`: Accepts a list of integers (the plate identifiers) to stack on top.
- `unstack(n: int)`: Removes n plates from the top of the stack. Returns the list of unstacked plates.
- `setCapacity(capacity: int)`: Sets a new capacity for the stack. If the current number of plates exceeds the new capacity, the stack must be reduced to fit the new limit by unstacking plates from the bottom.

Your class should handle edge cases such as:
- Trying to unstack more plates than are present.
- Setting a capacity lower than the current number of stacked plates.

EXAMPLES:
Input: 
```python
plate_stack = PlateStack()
plate_stack.setCapacity(5)
plate_stack.stack([1, 2, 3, 4])
```
Output: 
```python
None
```

```python
plate_stack.stack([5, 6, 7])
```
Output: 
```python

```

Output: 
```python
[7, 6, 5, 4]
```
Explanation: After unstacking 4 plates, plate identifiers 7, 6, 5, and 4 are removed.

```python
plate_stack.setCapacity(3)
```

```python
None
```

CONSTRAINTS:
- The plate identifiers are unique positive integers.
- The maximum number of plates that can be stacked at any time is 10^5.
- The operations `stack`, `unstack`, and `setCapacity` can be called in any order.

HINT:
Consider using an array to represent the stack and a pointer to manage the top of the stack efficiently.

SOLUTION_PYTHON:
```python
class PlateStack:
    def __init__(self):
        self.stack = []
        self.capacity = float('inf')  # No limit initially

    def stack(self, plates):
        for plate in plates:
            if len(self.stack) < self.capacity:
                self.stack.append(plate)

    def unstack(self, n):
        if n > len(self.stack):
            n = len(self.stack)  # Can't unstack more than available
        unstacked = []
        for _ in range(n):
            unstacked.append(self.stack.pop())
        return unstacked

    def setCapacity(self, capacity):
        self.capacity = capacity
        while len(self.stack) > capacity:
            self.unstack(1)  # Unstack one plate until within capacity

# Example Usage
plate_stack = PlateStack()
plate_stack.setCapacity(3)
plate_stack.stack([1, 2, 3, 4, 5])
print(plate_stack.unstack(2))  # Output: [5, 4]
```

SOLUTION_JS:
```javascript
class PlateStack {
    constructor() {
        this.stack = [];
        this.capacity = Infinity; // No limit initially
    }

    stack(plates) {
        for (let plate of plates) {
            if (this.stack.length < this.capacity) {
                this.stack.push(plate);
            }
        }
    }

    unstack(n) {
        n = Math.min(n, this.stack.length); // Can't unstack more than available
        let unstacked = [];
        for (let i = 0; i < n; i++) {
            unstacked.push(this.stack.pop());
        }
        return unstacked;
    }

    setCapacity(capacity) {
        this.capacity = capacity;
        while (this.stack.length > capacity) {
            this.unstack(1); // Unstack one plate until within capacity
        }
    }
}

// Example Usage
let plateStack = new PlateStack();
plateStack.setCapacity(3);
plateStack.stack([1, 2, 3, 4, 5]);
console.log(plateStack.unstack(2)); // Output: [5, 4]
```

TIME_COMPLEXITY: O(n) for stacking, O(n) for unstacking, O(n) for setting capacity, where n is the number of plates.
SPACE_COMPLEXITY: O(m) where m is the current number of plates in the stack. 

EXPLANATION:
The `PlateStack` class uses a simple array to represent the stack of plates. The `stack` method checks against the capacity before adding plates, while the `unstack` method retrieves and removes plates from the end of the array. The `setCapacity` method ensures that the current stack conforms to the new capacity, effectively unstacking as necessary to meet the limit while also maintaining efficiency with array operations.
