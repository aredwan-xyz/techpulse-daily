TITLE: Sum of Distinct Pairs
DIFFICULTY: Medium
TOPIC: Two Pointers
ESTIMATED_TIME: 30 minutes

PROBLEM:
Given an array of integers, you need to find the sum of distinct pairs of numbers that add up to a target sum. Each pair should only be counted once, and the order of numbers in a pair does not matter. Your goal is to return the sum of these distinct pairs. 

For example, if a pair (a, b) is found, it should be counted as the single value a + b. If an array has duplicates, each unique value should only contribute once to the final sum.

EXAMPLES:
Input: arr = [1, 2, 3, 4, 3, 5], target = 6
Output: 9
Explanation: The distinct pairs are (1, 5), (2, 4), and (3, 3). Their sums are 6, 6, and 6, respectively, leading to a total of 9.

Input: arr = [5, 4, 3, 3, 2, 1], target = 5
Output: 6
Explanation: The distinct pairs are (1, 4) and (2, 3). Their sums are 5 and 5, contributing to a total of 6.

CONSTRAINTS:
- The input array will have at most 1000 integers.
- Each integer will be in the range of -10^5 to 10^5.

HINT:
Consider sorting the array first to easily identify pairs.

SOLUTION_PYTHON:
```python
def sum_of_distinct_pairs(arr, target):
    # Sort the array to facilitate the two-pointer approach
    arr = sorted(set(arr))  # Removing duplicates and sorting
    left, right = 0, len(arr) - 1
    distinct_sums = set()

    while left < right:
        current_sum = arr[left] + arr[right]
        if current_sum == target:
            distinct_sums.add(current_sum)
            left += 1
            right -= 1
        elif current_sum < target:
            left += 1
        else:
            right -= 1

    return sum(distinct_sums)

# Time Complexity: O(n log n) for sorting + O(n) for the two-pointer traversal
# Space Complexity: O(n) due to the set for distinct values
```

SOLUTION_JS:
```javascript
function sumOfDistinctPairs(arr, target) {
    // Create a Set to remove duplicates
    let uniqueValues = Array.from(new Set(arr));
    // Sort the unique values
    uniqueValues.sort((a, b) => a - b);
    
    let left = 0;
    let right = uniqueValues.length - 1;
    let distinctSums = new Set();
    
    while (left < right) {
        let currentSum = uniqueValues[left] + uniqueValues[right];
        if (currentSum === target) {
            distinctSums.add(currentSum);
            left++;
            right--;
        } else if (currentSum < target) {
            left++;
        } else {
            right--;
        }
    }
    
    return Array.from(distinctSums).reduce((a, b) => a + b, 0);
}

// Time Complexity: O(n log n) for sorting + O(n) for two-pointer traversal
// Space Complexity: O(n) due to the set for distinct values
```

TIME_COMPLEXITY: O(n log n)
SPACE_COMPLEXITY: O(n)

EXPLANATION:
The approach begins by removing duplicates from the input array and sorting the unique values to apply the two-pointer technique efficiently. The two pointers start at either end of the sorted array, allowing us to find pairs that sum to the target while ensuring each pair is only counted once. The final result aggregates the unique sums created by valid pairs.
