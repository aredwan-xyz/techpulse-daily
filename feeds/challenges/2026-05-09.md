TITLE: Longest Substring with K Unique Characters
DIFFICULTY: Hard
TOPIC: Sliding Window
ESTIMATED_TIME: 45 minutes

PROBLEM:
You are given a string, `s`, and an integer `k`. Your task is to find the longest substring of `s` that contains exactly `k` unique characters. If there are multiple longest substrings, return the one that appears first in the string. If no such substring exists, return an empty string.

Examples:
Input: s = "abcabcbb", k = 2
Output: "ab"
Explanation: The longest substring with exactly 2 unique characters is "ab", starting at index 0.

Input: s = "aabbcc", k = 3
Output: "aabbcc"
Explanation: The entire string contains exactly 3 unique characters, thus it is the longest possible substring.

CONSTRAINTS:
- 1 <= |s| <= 10^5
- 1 <= k <= 26 (the number of unique characters in the standard English alphabet)

HINT:
Consider using a dictionary to track the count of characters in the current window.

SOLUTION_PYTHON:
```python
def longest_substring_k_unique(s: str, k: int) -> str:
    if k == 0 or not s:
        return ""
    
    left = 0
    max_length = 0
    start = 0
    char_count = {}
    
    for right in range(len(s)):
        # Add the current character to the dictionary
        char_count[s[right]] = char_count.get(s[right], 0) + 1
        
        # Shrink the window until we have exactly k unique characters
        while len(char_count) > k:
            char_count[s[left]] -= 1
            if char_count[s[left]] == 0:
                del char_count[s[left]]
            left += 1
        
        # Update the maximum length when exactly k unique characters
        if len(char_count) == k:
            if (right - left + 1) > max_length:
                max_length = right - left + 1
                start = left
    
    return s[start:start + max_length]
```

SOLUTION_JS:
```javascript
function longestSubstringKUnique(s, k) {
    if (k === 0 || s.length === 0) return '';
    
    let left = 0;
    let maxLength = 0;
    let start = 0;
    const charCount = {};
    
    for (let right = 0; right < s.length; right++) {
        // Add the current character to the hash map
        charCount[s[right]] = (charCount[s[right]] || 0) + 1;
        
        // Shrink the window until we have exactly k unique characters
        while (Object.keys(charCount).length > k) {
            charCount[s[left]]--;
            if (charCount[s[left]] === 0) {
                delete charCount[s[left]];
            }
            left++;
        }

        // Update the maximum length when exactly k unique characters
        if (Object.keys(charCount).length === k) {
            if ((right - left + 1) > maxLength) {
                maxLength = right - left + 1;
                start = left;
            }
        }
    }
    
    return s.substring(start, start + maxLength);
}
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(k)

EXPLANATION:
This approach employs a sliding window technique to track the number of unique characters within a substring. By expanding the right end of the window, we add characters to our count dictionary. If we exceed `k` unique characters, we increment the left end of the window to reduce the count until we are back to exactly `k`. This ensures that we can dynamically check the longest substring with the required properties in linear time.
