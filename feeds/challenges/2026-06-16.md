PROBLEM:
You are given a dataset containing customer purchase information for an online retail store. The dataset contains the following columns: `customer_id`, `age`, `annual_income`, `spending_score`, and `country`. Your task is to perform a customer segmentation analysis to identify different segments of customers based on their purchasing behavior using the K-means clustering algorithm. Finally, visualize the results in a scatter plot of income vs. spending score, coloring the points by their assigned cluster.

Write a Python function that processes the given dataset, applies K-means clustering to segment the customers, and returns both the modified dataset with a new `cluster_label` column and the scatter plot.

HINT:
Explore the elbow method to determine the optimal number of clusters for K-means.

SOLUTION_PYTHON:
```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

def customer_segmentation(data):
    # Create a DataFrame
    df = pd.DataFrame(data)

    # Select features for clustering
    features = df[['annual_income', 'spending_score']]

    # Standardizing the Features
    scaler = StandardScaler()
    scaled_features = scaler.fit_transform(features)

    # Using the elbow method to find the optimal number of clusters
    inertia = []
    for i in range(1, 11):
        kmeans = KMeans(n_clusters=i, random_state=0)
        kmeans.fit(scaled_features)
        inertia.append(kmeans.inertia_)

    # Choosing an optimal number of clusters (say 3 for this example) 
    optimal_clusters = 3
    kmeans = KMeans(n_clusters=optimal_clusters, random_state=0)
    df['cluster_label'] = kmeans.fit_predict(scaled_features)

    # Plotting the results
    plt.figure(figsize=(10, 6))
    plt.scatter(df['annual_income'], df['spending_score'], c=df['cluster_label'], cmap='viridis')
    plt.title('Customer Segmentation')
    plt.xlabel('Annual Income')
    plt.ylabel('Spending Score')
    plt.colorbar(label='Cluster Label')
    plt.show()

    return df

# Example usage:
data = {
    'customer_id': [1, 2, 3, 4, 5],
    'age': [22, 25, 47, 35, 26],
    'annual_income': [15000, 20000, 30000, 25000, 32000],
    'spending_score': [39, 81, 6, 77, 40],
    'country': ['US', 'US', 'UK', 'UK', 'US']
}
print(customer_segmentation(data))
```

// Using the TensorFlow.js library for K-means clustering
async function customerSegmentation(data) {
    const tf = require('@tensorflow/tfjs-node');
    const { KMeans } = require('kmeans-js');
    
    // Create tensors for annual income and spending score
    const income = data.map(d => d.annual_income);
    const spendingScore = data.map(d => d.spending_score);
    
    const features = income.map((inc, index) => [inc, spendingScore[index]]);
    
    // Initialize KMeans with a desired number of clusters
    const kmeans = new KMeans({ k: 3, maxIterations: 100 });
    const clusters = kmeans.cluster(features);
    
    // Return modified data
    data.forEach((d, index) => {
        d.cluster_label = clusters[index];
    });
    
    // Add visualization portion as needed using D3.js or similar
    
    return data;
}

The approach involves standardizing the features to improve the performance of K-means clustering. The elbow method helps to decide the optimal number of clusters to use. After clustering, the customer dataset is updated with the assigned cluster labels, and a scatter plot visualizes the segmentation based on annual income and spending scores, allowing for quick insights into customer behavior.
