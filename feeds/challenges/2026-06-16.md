TITLE: Temporal Data Interpolation and Averaging
DIFFICULTY: Hard
TOPIC: AI/ML Coding (NumPy/Pandas)
ESTIMATED_TIME: 45 minutes

PROBLEM:
You are tasked with analyzing a dataset containing time series data. The dataset consists of timestamps and corresponding value measurements from various sensors. Due to various reasons, certain time intervals may have missing data. Your job is to perform linear interpolation on the dataset to fill in these gaps and then calculate the average value of each sensor over the entire time period after interpolation. You need to create a function that takes in a Pandas DataFrame and returns a new DataFrame containing the interpolated values along with the average for each sensor.

Your input DataFrame will have the following structure:
- `timestamp`: Datetime index
- `sensor_id`: Unique identifier for each sensor
- `value`: Measurement values which may contain NaNs for missing data

Your output DataFrame should have the following structure:
- `sensor_id`: Unique identifier for each sensor
- `avg_value`: The average value of the sensor after interpolation

EXAMPLES:
Input:
```
| timestamp           | sensor_id | value |
|---------------------|-----------|-------|
| 2026-06-16 00:00:00 | S1        | 10.0  |
| 2026-06-16 00:05:00 | S1        | NaN   |
| 2026-06-16 00:10:00 | S1        | 15.0  |
| 2026-06-16 00:00:00 | S2        | 20.0  |
| 2026-06-16 00:05:00 | S2        | NaN   |
| 2026-06-16 00:10:00 | S2        | 30.0  |
```
Output:
```
| sensor_id | avg_value |
|-----------|-----------|
| S1        | 12.5      |
| S2        | 25.0      |
```
Explanation: For sensor S1, the missing value at 00:05:00 is interpolated to 12.5. The average for S1 is then (10.0 + 12.5 + 15.0) / 3 = 12.5. For sensor S2, the missing value is interpolated to 25.0, giving an average of (20.0 + 25.0 + 30.0) / 3 = 25.0.

Input:
```

|---------------------|-----------|-------|
| 2026-06-16 00:00:00 | S1        | NaN   |
| 2026-06-16 00:05:00 | S1        | 10.0  |
| 2026-06-16 00:10:00 | S1        | NaN   |
| 2026-06-16 00:00:00 | S2        | 40.0  |
| 2026-06-16 00:05:00 | S2        | NaN   |

```
Output:
```
| sensor_id | avg_value |
|-----------|-----------|
| S1        | 10.0      |
| S2        | 45.0      |
```
Explanation: For sensor S1, the missing values are interpolated to 10.0. Then the average is (10.0) = 10.0. For sensor S2, the missing value at 00:05:00 averages to 45.0.

CONSTRAINTS:
- The length of the DataFrame will be between 1 and 10,000 rows.
- Each sensor will have at least one valid measurement.

HINT:
Consider using the Pandas interpolation method to handle NaN values efficiently.

SOLUTION_PYTHON:
```python
import pandas as pd

def interpolate_and_average(df):
    # Ensure the timestamp is sorted and set as index
    df['timestamp'] = pd.to_datetime(df['timestamp'])
    df = df.sort_values(by=['sensor_id', 'timestamp']).set_index('timestamp')

    # Interpolating missing values for each sensor
    interpolated_df = df.groupby('sensor_id').apply(lambda group: group.interpolate(method='linear'))

    # Calculating the average value for each sensor after interpolation
    results = interpolated_df.groupby('sensor_id')['value'].mean().reset_index()
    results.columns = ['sensor_id', 'avg_value']
    
    return results

# Example usage:

```

SOLUTION_JS:
```javascript
const interpolateAndAverage = (df) => {
    // Convert DataFrame to an array of objects for processing
    const dfArray = df.toArray();
    const sensorMap = {};
    
    dfArray.forEach(({ timestamp, sensor_id, value }) => {
        if (!sensorMap[sensor_id]) {
            sensorMap[sensor_id] = { values: [], timestamps: [] };
        }
        sensorMap[sensor_id].values.push(value);
        sensorMap[sensor_id].timestamps.push(new Date(timestamp));
    });

    const avgResults = Object.keys(sensorMap).map(sensor_id => {
        const { values } = sensorMap[sensor_id];
        const interpolatedValues = interpolate(values); // Assume interpolate function performs linear interpolation.
        const avgValue = interpolatedValues.reduce((acc, val) => acc + val, 0) / interpolatedValues.length;
        return { sensor_id, avg_value: avgValue };
    });

    return avgResults;
};
```

SPACE_COMPLEXITY: O(n)

EXPLANATION:
The approach involves sorting the DataFrame, which takes O(n log n) time due to the need to arrange the data properly. We then employ `groupby` and `interpolate`, which are both linear operations on their respective groups, making it efficient for filling in the missing values. Finally, we aggregate the results to get the average for each sensor, resulting in optimal performance given the constraints.
