TITLE: Bit Counter
DIFFICULTY: Easy
TOPIC: Bit Manipulation
ESTIMATED_TIME: 15 minutes

PROBLEM:
Your task is to write a function that takes an integer as input and returns the count of '1' bits (also known as the Hamming weight) in its binary representation. This is useful in digital systems where it is important to assess the number of active bits in a number.

EXAMPLES:
Input: 5
Output: 2
Explanation: The binary representation of 5 is `101`, which has two '1' bits.

Input: 13
Output: 3
Explanation: The binary representation of 13 is `1101`, which has three '1' bits.

CONSTRAINTS:
- The input integer will be a non-negative integer up to 2^31 - 1.
- Your solution must run in O(log n) time, where n is the input integer.

HINT:
You can repeatedly check if the least significant bit is a '1' and use bitwise right shift to process the next bit.

SOLUTION_PYTHON:
```python
def count_one_bits(n: int) -> int:
    count = 0
    while n > 0:
        count += n & 1  # Check if the least significant bit is a '1'
        n >>= 1        # Right shift to process the next bit
    return count

# Example usage
print(count_one_bits(5))  # Output: 2
print(count_one_bits(13)) # Output: 3
```

SOLUTION_JS:
```javascript
function countOneBits(n) {
    let count = 0;
    while (n > 0) {
        count += n & 1; // Check if the least significant bit is a '1'
        n >>= 1;       // Right shift to process the next bit
    }
    return count;
}

// Example usage
console.log(countOneBits(5));  // Output: 2
console.log(countOneBits(13)); // Output: 3
```

TIME_COMPLEXITY: O(log n)
SPACE_COMPLEXITY: O(1)

EXPLANATION:
The approach leverages bit manipulation to efficiently count the number of '1' bits in the integer. By checking the least significant bit using the bitwise AND operator and then right-shifting the number, you can count how many bits are set to '1'. This is optimal as it only processes each bit once, resulting in a logarithmic time complexity relative to the size of the number.
