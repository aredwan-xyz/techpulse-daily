PROBLEM:
You are tasked with analyzing customer data from an e-commerce platform to help design personalized marketing strategies. The dataset contains information about customer purchases, demographics, and browsing behavior. Your goal is to segment customers into distinct groups based on their purchasing habits and feature similarities using the K-means clustering algorithm.

The dataset is structured as follows:
- `customer_id`: Unique identifier for each customer.
- `age`: Age of the customer.
- `annual_income`: Annual income of the customer.
- `spending_score`: A score assigned to the customer based on their spending behavior (ranging from 1 to 100).
- `purchase_frequency`: Number of purchases made per month.

Your task is to:
1. Load the dataset into a Pandas DataFrame.
2. Normalize the numerical features.
3. Apply the K-means clustering algorithm to segment customers into clusters (you may choose the optimal number of clusters using the elbow method).
4. Return the cluster centers and the number of customers in each cluster.

Consider using the `KMeans` implementation from Scikit-learn along with the `elbow method` to determine the optimal number of clusters.

SOLUTION_PYTHON:
```python
import pandas as pd
import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

def customer_segmentation(data):
    # Load data into DataFrame
    df = pd.DataFrame(data)
    
    # Extract features
    features = df[['age', 'annual_income', 'spending_score', 'purchase_frequency']]
    
    # Normalize the features
    scaler = StandardScaler()
    features_normalized = scaler.fit_transform(features)
    
    # Determine the optimal number of clusters using Elbow method
    inertia = []
    K = range(1, 11)  # Example range for clusters
    for k in K:
        kmeans = KMeans(n_clusters=k, random_state=0)
        kmeans.fit(features_normalized)
        inertia.append(kmeans.inertia_)
    
    # Plotting the elbow curve
    plt.plot(K, inertia, 'bx-')
    plt.xlabel('Number of clusters')
    plt.ylabel('Inertia')
    plt.title('Elbow Method For Optimal k')
    plt.show()
    
    # Assuming from the plot, we choose k=2 (this can be adjusted)
    optimal_k = 2
    kmeans = KMeans(n_clusters=optimal_k, random_state=0)
    kmeans.fit(features_normalized)
    
    # Getting cluster centers and sizes
    cluster_centers = scaler.inverse_transform(kmeans.cluster_centers_)
    cluster_labels = kmeans.labels_
    cluster_sizes = np.bincount(cluster_labels)
    
    return {
        'cluster_centers': cluster_centers.tolist(),
        'cluster_sizes': cluster_sizes.tolist()
    }
```

```javascript
import * as dfd from "danfojs-node";
import { KMeans } from "ml-kmeans";

async function customerSegmentation(data) {
    const df = new dfd.DataFrame(data);
    
    // Extract features and normalize
    const features = df.loc({ columns: ['age', 'annual_income', 'spending_score', 'purchase_frequency'] });
    const normalizedFeatures = features.mul(1 / (features.max().values)); // Simple normalization

    // Determine the optimal number of clusters using the Elbow method
    const inertia = [];
    const maxK = 10; 
    for (let k = 1; k < maxK; k++) {
        const kmeans = new KMeans({ k: k });
        const clusters = kmeans.fit(normalizedFeatures.values);
        inertia.push(kmeans.inertia);
    }

    // Assuming we choose optimalK as 2 based on the elbow method
    const optimalK = 2;
    const kmeansFinal = new KMeans({ k: optimalK });
    const finalClusters = kmeansFinal.fit(normalizedFeatures.values);
    
    const clusterCenters = kmeansFinal.centroids;
    const clusterSizes = Array.from({ length: optimalK }, (_, i) => finalClusters.labels.filter(label => label === i).length);
    
    return {
        cluster_centers: clusterCenters,
        cluster_sizes: clusterSizes
    };
}
```

TIME_COMPLEXITY: O(n * k * t) where n is the number of data points, k is the number of clusters, and t is the number of iterations to converge.
SPACE_COMPLEXITY: O(n + k)

EXPLANATION:
To approach this problem, we first normalize the features to ensure they are on a similar scale which helps in obtaining a more accurate clustering. We then utilize the Elbow method to determine the optimal number of clusters by plotting the inertia associated with different cluster counts. Finally, we fit the K-means model to our normalized data and extract the resulting cluster centers and sizes. The methodology effectively highlights customer segments for customized marketing strategies.
