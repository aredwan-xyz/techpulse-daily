TITLE: Temporal Pattern Detection in Time Series Data  
DIFFICULTY: Hard  
TOPIC: AI/ML Coding (NumPy/Pandas)  
ESTIMATED_TIME: 60 minutes  

PROBLEM:  
You are given a time series dataset represented as a Pandas DataFrame that contains two columns: a timestamp and a numerical value recorded at that timestamp. Your task is to implement a function that identifies any repeating patterns (cycles) in the data for a given period 'n'. The function should return the starting timestamp of each detected cycle standardizing it to the corresponding starting timestamp of the recurring patterns.

The DataFrame's timestamps are guaranteed to be sorted and without gaps. A cycle is defined as a segment of the dataset that repeats continuously over any number of periods defined by 'n'.

EXAMPLES:  
Input:  
```python
import pandas as pd

data = {
    'timestamp': pd.date_range(start='2026-05-01', periods=10, freq='D'),
    'value': [1, 2, 3, 1, 2, 3, 1, 2, 3, 1]
}
df = pd.DataFrame(data)

n = 3
```
Output:  
```python
['2026-05-01', '2026-05-04', '2026-05-07']
```
Explanation:  
The values [1, 2, 3] repeat every 3 days starting from '2026-05-01', '2026-05-04', and '2026-05-07'.

Input:  
```python
data = {
    'timestamp': pd.date_range(start='2026-05-01', periods=6, freq='D'),
    'value': [1, 1, 1, 1, 2, 2]
}
n = 2
```
Output:  
```python
[]
```
Explanation:  
There are no repeating patterns of length 2 in the value column.

CONSTRAINTS:  
- The length of the DataFrame will be at least `2` and at most `10^4`.  
- The value column will contain numerical values that the function needs to evaluate for cyclicity.

HINT:  
Consider utilizing a sliding window technique to compare subsequences in the DataFrame.

SOLUTION_PYTHON:
```python
import pandas as pd

def detect_cycles(df: pd.DataFrame, n: int) -> list:
    timestamps = df['timestamp'].values
    values = df['value'].values
    
    detected_cycles = []
    length = len(values)
    
    for start in range(length - n):
        segment = values[start:start + n]
        # Check subsequent segments
        for i in range(start + n, length - n + 1, n):
            if (values[i:i + n] == segment).all():
                # Save the corresponding timestamp of the starting segment
                detected_cycles.append(timestamps[start])
                break  # To avoid duplicates per detected pattern
    
    return detected_cycles

# Time complexity: O(m * k), where m is the total length of the series and k is the number of cycles checked.
# Space complexity: O(c), where c is the number of detected cycles.
```

SOLUTION_JS:
```javascript
function detectCycles(df, n) {
    let detectedCycles = [];
    const values = df.value;
    const timestamps = df.timestamp;
    const length = values.length;

    for (let start = 0; start <= length - n; start++) {
        const segment = values.slice(start, start + n);
        
        for (let i = start + n; i <= length - n; i += n) {
            const nextSegment = values.slice(i, i + n);
            if (JSON.stringify(nextSegment) === JSON.stringify(segment)) {
                detectedCycles.push(timestamps[start]);
                break; // to avoid duplicates
            }
        }
    }
    
    return detectedCycles;
}

// Time complexity: O(m * k), where m is the total length of the series and k is the number of cycles checked.
// Space complexity: O(c), where c is the number of detected cycles.
```

TIME_COMPLEXITY: O(m * k)  
SPACE_COMPLEXITY: O(c)  

EXPLANATION:  
The solution employs a nested loop structure where the outer loop iterates through potential starting indices of the segments and the inner loop checks for repeated patterns by comparing the current segment against the future segments. This approach effectively detects cycles while maintaining clarity and efficiency.
