PROMPT_TITLE: Clear Code Refactoring Wizard

THE_PROMPT:
You are a refactoring wizard tasked with improving the clarity and maintainability of the given code. Please analyze the following code snippet {CODE_SNIPPET} and provide a refactored version that enhances readability and structure without altering its functionality. Focus on the following improvements: 
1. Use meaningful variable names.
2. Break down complex functions into smaller, more manageable ones.
3. Add comments to clarify the purpose of key sections.
4. Remove any redundant or unused code.
5. Ensure adherence to best practices and coding standards.

Additionally, explain your refactoring choices in a brief summary highlighting the benefits of each change made.

WHEN_TO_USE:
Use this prompt when you have a code snippet that is working but hard to read or maintain. It’s particularly useful for situations where team collaboration or future updates are expected, making clarity essential.

EXAMPLE_INPUT:
{CODE_SNIPPET}: 
```
function doStuff(x, y) {
    if (x > 10) {
        for (var i = 0; i < y; i++) {
            console.log("Processing " + i);
        }
    }
}
```

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces a clean, well-structured version of the original code with meaningful variable names, enhanced organization, and explanatory comments. It also includes a summary of the changes made, clarifying the rationale behind each adjustment.

PRO_TIPS:
- Provide a clear and specific code snippet to ensure targeted refactoring.
- Highlight any specific areas of concern within the code that you want addressed.
- Specify any coding standards or languages that should be adhered to for better relevance to your project.
