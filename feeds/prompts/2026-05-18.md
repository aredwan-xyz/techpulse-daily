PROMPT_TITLE: Comprehensive Test Suite Generator

THE_PROMPT:
Create a comprehensive test suite for the following software component. Include test cases for various scenarios, covering edge cases, error handling, and performance testing. Please provide the test suite in a structured format, including the following sections: 

1. **Test Case ID**: Unique identifier for each test case.
2. **Description**: Brief explanation of what the test case will validate.
3. **Preconditions**: Any necessary conditions that must be met before executing the test.
4. **Inputs**: Specific inputs or parameters needed to run the test.
5. **Expected Results**: Detailed description of what the expected outcome of the test is.
6. **Execution Steps**: A step-by-step guide on how to execute the test.

Use the following details to generate the test suite:
- **Component Name**: {COMPONENT_NAME}
- **Functionality**: {FUNCTIONALITY_DESCRIPTION}
- **Specific Scenarios**: {SCENARIO_DESCRIPTIONS}

WHEN_TO_USE:
This prompt is ideal when you need to create a detailed and structured test suite for a new software feature or when revising existing tests for thoroughness. It ensures that all vital scenarios, including edge cases and performance aspects, are considered.

EXAMPLE_INPUT:
- COMPONENT_NAME: User Authentication Module
- FUNCTIONALITY_DESCRIPTION: Handle user login and logout processes securely
- SCENARIO_DESCRIPTIONS: Successful login, Failed login due to incorrect password, Login attempt with locked account, Successful logout

EXAMPLE_OUTPUT_SUMMARY:
The output will be a well-organized test suite that includes multiple test cases for the user authentication module. Each test case will clearly specify the steps to execute, expected results, and preconditions, ensuring thorough coverage for the given functionality.

PRO_TIPS:
- Clearly define the functionality to avoid ambiguity in the generated test cases.
- Include a variety of edge cases and negative scenarios to ensure robust testing.
- Review the generated test cases for completeness and clarity to enhance utility. 

WORKS_WITH: Claude · GPT-4 · Gemini
