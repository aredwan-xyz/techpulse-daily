PROMPT_TITLE: Unleash Performance Potential

THE_PROMPT:
You are a performance optimization expert tasked with identifying and resolving performance bottlenecks in a {APPLICATION_NAME} application. Begin by analyzing the application's overall architecture and pinpointing areas where inefficiencies may exist. Prepare a structured assessment that includes:
1. **Current Performance Metrics**: Provide key metrics like response time, throughput, and resource utilization.
2. **Identified Bottlenecks**: List specific components causing sluggish performance, such as database queries, API latency, or front-end rendering issues.
3. **Recommended Solutions**: For each bottleneck, suggest actionable strategies or optimizations, including best practices like caching, load balancing, code refactoring, or upgrading infrastructure.
4. **Expected Impact**: Offer insights on how the recommended solutions could improve performance metrics and user experience.
Format your findings in a clear, organized report aimed at both technical and non-technical stakeholders.

WHEN_TO_USE:
Use this prompt when you need to dissect the performance of an application and offer targeted solutions to enhance its efficiency. Ideal for software developers, system administrators, or project managers looking to optimize performance before product launches or major updates.

EXAMPLE_INPUT:
APPLICATION_NAME: Online Retail Store

EXAMPLE_OUTPUT_SUMMARY:
This prompt produces a detailed performance report that highlights key issues and provides a roadmap for optimizing application performance. It will outline current metrics, pinpoint performance bottlenecks, and present clear, actionable recommendations along with their expected benefits.

PRO_TIPS:
- Provide as much context about the application as possible for more tailored suggestions.
- Include specific performance metrics you're currently tracking for deeper insights.
- Consider the user experience perspective when evaluating and recommending solutions.
