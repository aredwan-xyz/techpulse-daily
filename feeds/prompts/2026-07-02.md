PROMPT_TITLE: Scalable System Blueprint Generator

THE_PROMPT:
You are an expert system designer tasked with creating a scalable system. Please provide a detailed architecture for the {SYSTEM_TYPE} that can handle {USER_LOAD} users efficiently. Include the following elements in your design:
1. **Overview**: A brief description of the system's purpose.
2. **Architecture Diagram**: Outline the main components and their interactions.
3. **Technology Stack**: List the technologies and tools you recommend for implementing each component.
4. **Scalability Strategies**: Describe how you would ensure the system can scale horizontally and vertically as needed.
5. **Data Flow**: Explain how data will be handled, stored, and accessed within the system.
6. **Monitoring & Maintenance**: Discuss your plans for monitoring system performance and maintenance strategies after deployment.

WHEN_TO_USE:
Use this prompt when you need to design a comprehensive, scalable system architecture, particularly for applications expected to grow in user base and complexity. It's ideal for startups and enterprises looking to leverage best practices in system design.

EXAMPLE_INPUT:
SYSTEM_TYPE: E-commerce platform  
USER_LOAD: 100,000  

EXAMPLE_OUTPUT_SUMMARY:
This prompt produces a well-structured system design plan that includes an overview and a detailed architecture diagram, along with recommendations for a technology stack, scalability strategies, data handling processes, and maintenance plans. It's tailored to support strategic decision-making and development processes.

PRO_TIPS:
- Specify user load projections to guide the scalability aspect more precisely.
- Include any specific technologies you prefer or constraints that must be considered.
- Outline expected features or functionalities to refine the overall design.
