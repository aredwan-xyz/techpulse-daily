PROMPT_TITLE: Scalable System Blueprint Builder

THE_PROMPT:
Design a scalable {SYSTEM_TYPE} that meets the following requirements:
1. **Purpose & Goals**: Describe the primary goals of the system and the key functionalities it must support (e.g., user base expansion, data processing, etc.)
2. **User Demographics**: Identify the target users and expected user traffic for this system, including peak load estimations.
3. **Key Components**: List out the main components of the system (e.g., databases, servers, APIs) and specify their roles in ensuring scalability.
4. **Technology Stack**: Suggest appropriate technologies and frameworks for each component that align with the system's needs (e.g., programming languages, database systems, cloud services).
5. **Scalability Strategies**: Outline strategies to handle scalability including load balancing, caching solutions, and database replication strategies.
6. **Potential Challenges**: Discuss potential bottlenecks and challenges the system might face as usage scales up, along with proposed solutions.

WHEN_TO_USE:
Use this prompt when you need to create a detailed and scalable system design for a specific application or project. It's ideal for system architects, product managers, and developers looking to establish a strong foundation for future growth.

EXAMPLE_INPUT:
SYSTEM_TYPE: E-commerce platform
Goals: Increase user engagement and reduce latency in transactions.
Target Users: 10,000 concurrent users during peak times.

EXAMPLE_OUTPUT_SUMMARY:
This prompt generates a comprehensive blueprint for designing a scalable system, including essential components, technology choices, scalability strategies, and considerations for challenges. Users receive a structured plan to enhance system performance and user experience.

PRO_TIPS:
- Be as specific as possible when defining goals to ensure the design aligns well with your vision.
- Consider the potential growth in user numbers or data volume when detailing scalability strategies.
- Review similar existing systems for comparative analysis and inspiration.
