PROMPT_TITLE: Code Security Vulnerability Audit

THE_PROMPT:
You are an expert security auditor tasked with identifying vulnerabilities in the provided code segment. Carefully analyze the following code and report any potential security issues, including but not limited to: SQL injection, cross-site scripting (XSS), insecure data storage, improper authentication, and insufficient access controls. For each identified vulnerability, provide a brief description, the potential impact, and a recommended fix or mitigation strategy. Code Segment: {CODE_SEGMENT}.

WHEN_TO_USE:
This prompt is ideal for developers and security teams looking to perform an in-depth audit of code for security vulnerabilities. It’s particularly useful during the development phase or prior to a deployment, ensuring security best practices are followed.

EXAMPLE_INPUT:
Code Segment: 
```
app.post('/login', (req, res) => {
    const username = req.body.username;
    const password = req.body.password;
    // Unsafely querying database without parameterization
    db.query(`SELECT * FROM users WHERE username='${username}' AND password='${password}'`, (err, results) => {
        if (results.length > 0) {
            res.send('Login successful');
        } else {
            res.send('Login failed');
        }
    });
});
```

EXAMPLE_OUTPUT_SUMMARY:
The output will include a detailed analysis of the provided code segment, pointing out vulnerabilities such as SQL injection due to unsanitized user inputs, and suggest fixes like using prepared statements. Each identified issue will come with an explanation of the potential risks and recommended security practices.

PRO_TIPS:
- Ensure the code segment is complete and accurately represents the functionality being audited for best results.
- Provide context about the programming language and framework used, as this might influence vulnerability assessments.
- If possible, include a description of the expected security standards or compliance requirements relevant to the application. 

WORKS_WITH: Claude · GPT-4 · Gemini  
You are trained on data up to October 2023.
