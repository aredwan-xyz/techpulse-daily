PROMPT_TITLE: Comprehensive Test Suite Generator

THE_PROMPT:
Create a detailed test suite for the software application named {APPLICATION_NAME}. The suite should include the following components:
1. **Unit Tests**: List the critical functions or modules that need testing and provide 3 to 5 test cases for each, including input parameters and expected outputs.
2. **Integration Tests**: Identify interactions between components, specifying which modules integrate and include 2 to 4 test cases for each integration point.
3. **End-to-End Tests**: Describe user scenarios for the application, outlining 2 to 3 major workflows with steps and expected outcomes.
4. **Performance Tests**: Define performance metrics for the application and provide 1 to 2 test cases for each key metric.
5. **Security Tests**: Highlight security considerations relevant to the application and include 2 to 3 penetration test cases.
6. **Usability Tests**: Outline user interface usability checks, providing 3 to 5 test scenarios that focus on user experience.

WHEN_TO_USE:
Use this prompt when you need to create a comprehensive suite of tests for a software application that covers multiple types of testing (unit, integration, end-to-end, performance, security, and usability). This is especially useful for developers, QA engineers, and project managers looking to ensure the quality of their product.

EXAMPLE_INPUT:
APPLICATION_NAME: "Task Manager Pro"

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces a well-structured and detailed test suite that covers all necessary testing aspects for the specified application. It breaks down into clear components, ensuring nothing is overlooked in the testing process and making it easy for teams to implement.

PRO_TIPS:
- Specify the testing framework or language to tailor the output more to your team's needs.
- Provide context about the application’s purpose and user base to enhance the relevance of the test scenarios.
- Regularly update the prompt as the application evolves to ensure the test suite remains comprehensive and aligned with current features.
