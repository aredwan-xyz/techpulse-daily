PROMPT_TITLE: Comprehensive Test Suite Generator

THE_PROMPT:
Generate a comprehensive test suite for the {APPLICATION_NAME} application focused on the {FEATURE_NAME} feature. The test suite should include the following components:
1. **Unit Tests**: Define specific unit tests targeting the core functionalities, including edge cases and expected outcomes.
2. **Integration Tests**: Describe integration scenarios with other features or services, ensuring robust interactions between components.
3. **Performance Tests**: Outline performance expectations, including load testing metrics and expected response times under varying conditions.
4. **Security Tests**: List security vulnerabilities to test against, including authentication, authorization, and data protection measures.
5. **Usability Tests**: Suggest methods for evaluating user experience, including accessibility considerations and user feedback methods.

Ensure each component is actionable, with clear instructions and criteria for success. 

WHEN_TO_USE:
This prompt is ideal for developers and QA testers looking to generate detailed test suites tailored to specific features of an application. It helps ensure comprehensive coverage of critical aspects, from functionality to performance and security.

EXAMPLE_INPUT:
APPLICATION_NAME: E-Commerce Site
FEATURE_NAME: Shopping Cart

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces a structured test suite that covers unit, integration, performance, security, and usability testing for the specified feature. Each section includes actionable test cases and evaluation criteria to facilitate effective testing.

PRO_TIPS:
- Be as specific as possible when defining {APPLICATION_NAME} and {FEATURE_NAME} to tailor the tests accurately.
- Include any relevant constraints or dependencies that may impact the testing logic.
- Consider adding expected outcomes for each test for clearer benchmarks during implementation.
