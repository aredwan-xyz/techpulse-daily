PROMPT_TITLE: Senior Engineer Code Review Request

THE_PROMPT:
You are a senior engineer with extensive experience in {PROGRAMMING_LANGUAGE}. I need your help reviewing my code for {PROJECT_NAME}. The primary goal of this review is to identify any potential issues, suggest improvements, and ensure the code adheres to best practices. Here are the coding details: 

1. **Code Snippet**: 
   ```{INSERT_CODE_SNIPPET}```
2. **Specific Areas of Focus**: 
   - Performance optimization
   - Readability and maintainability
   - Adherence to coding standards
3. **Context**: 
   - This code is intended for {FUNCTION or FEATURE} in the {APPLICATION_NAME} project.
   - The project is using {FRAMEWORK/LIBRARY}.

Please provide your feedback in a structured format, addressing each area of focus with comments and suggestions for improvement.

WHEN_TO_USE:
Use this prompt when you have written a piece of code and are looking to get in-depth feedback from a senior engineer. It is ideal for complex projects where adherence to best practices and optimization are crucial.

EXAMPLE_INPUT:
- PROGRAMMING_LANGUAGE: Python
- PROJECT_NAME: WeatherApp
- INSERT_CODE_SNIPPET: 
  ```python
  def get_weather_data(city):
      response = requests.get(f"https://api.weather.com/v3/weather/{city}")
      return response.json()
  ```
- FUNCTION or FEATURE: fetching weather data
- APPLICATION_NAME: WeatherApp
- FRAMEWORK/LIBRARY: Flask

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces a detailed code review that includes insights on performance improvements, suggestions for better readability, adherence to standards, and any potential bugs identified in the code provided.

PRO_TIPS:
- Always include a small, representative code snippet that can function independently for better feedback.
- Specify any project-specific constraints or standards to ensure targeted suggestions.
- Be open to criticism and ask for specific examples to clarify any suggested improvements.
