PROMPT_TITLE: Identify and Resolve Performance Bottlenecks

THE_PROMPT:
You are a performance optimization expert tasked with identifying and fixing performance bottlenecks in a {APPLICATION_TYPE} system. Please analyze the following information: 
- System architecture: {ARCHITECTURE_DESCRIPTION} 
- Current performance metrics: {METRICS} 
- User feedback and reported issues: {FEEDBACK} 
- Existing optimization efforts: {OPTIMIZATION_EFFORTS} 
Based on this information, provide a detailed assessment of potential bottlenecks and specific recommendations for enhancements. Include the following:
1. A brief overview of each identified bottleneck.
2. Suggested tools or techniques for analyzing performance further.
3. Concrete steps to implement the recommended optimizations, including any code or configuration examples. 

WHEN_TO_USE:
Use this prompt when you need to systematically identify and address performance issues within any application. It's ideal for developers, system architects, or anyone responsible for maintaining application performance. 

EXAMPLE_INPUT:
APPLICATION_TYPE: web application  
ARCHITECTURE_DESCRIPTION: microservices architecture with a Node.js backend and React frontend  
METRICS: average load time 5 seconds, CPU usage at 85% during peak hours  
FEEDBACK: users report slow page responses and frequent timeouts  
OPTIMIZATION_EFFORTS: previously optimized database queries but still facing performance issues

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces a structured analysis of the application's performance bottlenecks, providing clear recommendations for improvements. The output includes specific identification of issues, analytical tools suggestions, and actionable steps for developers to implement fixes.

PRO_TIPS:
- Include as much detail as possible in the input for more tailored outputs.
- Specify any particular constraints or requirements for the optimizations you are considering.
- Provide information about the expected user load to help tailor performance recommendations effectively.
