PROMPT_TITLE: Unlock Your Application's Best Performance

THE_PROMPT:  
As a software engineer, I am focused on enhancing the performance of my application. I need to identify and address performance bottlenecks. Here’s the context: {APPLICATION_NAME} is experiencing issues such as {PROBLEM_DESCRIPTIONS}. I would like you to analyze various aspects, including the code efficiency, resource utilization, and response times. Please provide specific recommendations for optimization strategies that can be applied to these areas: {LIST_OF_COMPONENTS}. Include code snippets for suggested improvements if applicable, and any tools you recommend for measuring and monitoring performance improvements.

WHEN_TO_USE:  
Use this prompt when you suspect that your application is sluggish or not performing to its potential. It can help pinpoint inefficiencies and recommend actionable strategies to enhance performance.

EXAMPLE_INPUT:  
APPLICATION_NAME: MyWebApp  
PROBLEM_DESCRIPTIONS: slow loading times, high CPU usage  
LIST_OF_COMPONENTS: front-end code, database queries, server configuration

EXAMPLE_OUTPUT_SUMMARY:  
This prompt will yield a detailed analysis of the current performance issues accompanied by tailored suggestions for optimization. Users can expect actionable insights relating to code changes, database indexing, and possible caching strategies, supported by practical examples and recommended performance monitoring tools.

PRO_TIPS:  
- Be as specific as possible when describing the problems and components to ensure targeted recommendations.  
- Provide existing performance metrics to give context for the optimization efforts.  
- Consider including any recent changes that might have influenced performance to narrow down potential issues.
