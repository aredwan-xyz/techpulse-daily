PROMPT_TITLE: Senior Engineer Code Review Request

THE_PROMPT:
"Hello [Senior Engineer's Name],

I would appreciate your insights on the following piece of code I’ve developed. The details are as follows:

**Context:**
- Project Name: {PROJECT_NAME}
- Purpose of the Code: {CODE_PURPOSE}

**Code Snippet:**
```python
{CODE_SNIPPET}
```

**Key Areas for Review:**
1. Code Efficiency - Are there any areas that could be optimized for performance?
2. Readability - Is the code easily understandable for future developers?
3. Best Practices - Have I adhered to the best coding standards and practices in this implementation?
4. Potential Bugs - Are there any potential issues or edge cases that I've missed?

If you could provide feedback on these points, I would greatly appreciate it. 

Thank you for your time!"

WHEN_TO_USE:
Use this prompt when you want a constructive and detailed code review from a senior engineer. It's especially valuable for complex or critical code segments where guidance can prevent future issues and improve overall quality.

EXAMPLE_INPUT:
Senior Engineer's Name: "Alex"
PROJECT_NAME: "Invoice Processing System"
CODE_PURPOSE: "To calculate and apply discounts to invoices based on customer tier."
CODE_SNIPPET: "def apply_discount(invoice_total, customer_tier): ..."

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably generates comprehensive feedback that covers multiple critical aspects of code quality, including efficiency, readability, adherence to best practices, and potential bugs, leading to improved code health and developer learning.

PRO_TIPS:
- Provide as much context as possible to help the reviewer understand the larger scope of the code.
- Highlight specific areas where you feel uncertain or wish to receive focused feedback.
- Keep the code snippet concise and relevant to the review to encourage thorough but time-efficient responses.
