PROMPT_TITLE: Comprehensive Test Suite Generator

THE_PROMPT:  
Generate a detailed test suite for the {APPLICATION_NAME} application. The test suite should cover various aspects including unit tests, integration tests, and end-to-end tests. Provide the following sections:  
1. **Test Objectives:** Brief overview of the primary goals for each test type.  
2. **Test Cases:** List specific test cases with clear input and expected output. Organize them by functionality or module.  
3. **Preconditions:** Detail any necessary setup or data that must exist before tests run.  
4. **Execution Steps:** Step-by-step instructions on how to run each test case.  
5. **Postconditions:** Expected state of the application or environment post-testing.  
Make sure to include edge cases and common user interactions. Overall, ensure the tests promote code quality and application reliability.

WHEN_TO_USE:  
Use this prompt when you need to generate a comprehensive test suite for an application under development or during maintenance phases. It's particularly helpful for QA engineers or developers aiming to enhance testing coverage.

EXAMPLE_INPUT:  
APPLICATION_NAME: MyECommerceApp

EXAMPLE_OUTPUT_SUMMARY:  
This prompt reliably produces a well-structured test suite including a range of test cases, a clear set of objectives for each test, execution steps, and expected outcomes. The output ensures thorough testing of the application to uphold quality and performance standards.

PRO_TIPS:  
- Ensure to provide detailed information about the application's functionalities to generate more tailored tests.  
- Consider different user roles during test case creation to cover various access levels and actions.  
- Regularly update the test suite to reflect changes in application features or user requirements.
