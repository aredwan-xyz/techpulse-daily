PROMPT_TITLE: Comprehensive Test Suite Generator

THE_PROMPT:
Generate a comprehensive test suite for the {APPLICATION_NAME} application. This test suite should include the following categories: unit tests, integration tests, functional tests, and performance tests. For each test category, provide at least {NUMBER} detailed test cases that cover various scenarios. Include preconditions, testing steps, expected outcomes, and any necessary test data. Ensure that the test cases address edge cases and potential failure points. If applicable, mention dependencies between tests or any specific tools that should be used for testing. The objective is to create a robust and systematic approach to validate all functionalities of the application.

WHEN_TO_USE:
Use this prompt when you need to create rigorous testing methodologies for a specific application. It's ideal for project managers, developers, and QA engineers who want to ensure their software is thoroughly vetted before release.

EXAMPLE_INPUT:
APPLICATION_NAME: "E-commerce Website"
NUMBER: 5

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces detailed and structured test cases across multiple testing categories. Each case outlines the essential components needed to verify the functionalities of the application, making it easy to identify both successes and failures during the testing phase.

PRO_TIPS:
- Clearly define the scope of your application to tailor the test cases to its specific functionalities.
- Use real-world scenarios when designing the test cases to enhance their applicability and relevance.
- Ensure that you include clear and concise expected outcomes to facilitate accurate assessments during testing.
