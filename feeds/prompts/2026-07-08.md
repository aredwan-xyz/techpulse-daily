PROMPT_TITLE: Trade-Off Tuner

THE_PROMPT:
You are a seasoned architect tasked with evaluating the technical trade-offs for a project. Consider the following details and provide a comprehensive analysis:

1. Project Overview: {Brief description of the project}
2. Key Requirements: {List of primary requirements}
3. Technical Options: {List of potential technical solutions or approaches}
4. Evaluation Criteria: {Key criteria to evaluate trade-offs, e.g., cost, scalability, performance, maintainability}
5. Constraints: {Any known limitations or constraints}
   
Based on these inputs, analyze each technical option against the evaluation criteria. Discuss the pros and cons, focusing on how each option aligns with the requirements and constraints outlined. Conclude your evaluation with a clear recommendation on which option is optimal for the project, justifying your choice based on the analysis.

WHEN_TO_USE:
Use this prompt when you need a structured approach for evaluating technical trade-offs in architecture decisions. It is particularly beneficial during the project planning phase or when considering a significant change in technology.

EXAMPLE_INPUT:
- Project Overview: Development of a cloud-based inventory management system
- Key Requirements: High scalability, real-time updates, user-friendly interface
- Technical Options: Microservices architecture, Monolithic architecture, Serverless architecture
- Evaluation Criteria: Cost, scalability, performance, maintainability
- Constraints: Limited budget and tight project timeline

EXAMPLE_OUTPUT_SUMMARY:
This prompt will yield a detailed trade-off analysis that highlights the strengths and weaknesses of each architectural approach. Users will receive a well-reasoned recommendation tailored to the specific project's needs, backed by a structured evaluation of the key criteria.

PRO_TIPS:
- Be specific in your project overview to guide the analysis effectively.
- Include as many evaluation criteria as possible to ensure a comprehensive evaluation.
- Highlight any critical constraints that may impact the recommendations significantly.
