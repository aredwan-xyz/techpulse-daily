PROMPT_TITLE: Senior Engineer Code Review Request

THE_PROMPT:
As a junior software engineer, I am seeking your expertise to review my code. Please find the details below:

- **Code Purpose**: {Briefly describe what the code is intended to do.}
- **Language/Framework**: {Specify the programming language and any frameworks used.}
- **Key Functionality**: {Summarize the main functions or classes in the code. Highlight any complex logic or algorithms.}
- **Specific Concerns**: {List any particular areas of the code you would like feedback on, such as performance, readability, or adherence to best practices.}
- **Repository Link**: {Include a link to the code repository, if applicable.}

Your insights on improving the code quality and best practices would be greatly appreciated!

WHEN_TO_USE:
Use this prompt when you're looking for a detailed and constructive review from a senior engineer on your code. It’s especially useful when you have specific areas of concern or complex implementations that need expert guidance.

EXAMPLE_INPUT:
- Code Purpose: Implementing a user authentication system
- Language/Framework: JavaScript with Node.js and Express
- Key Functionality: Auth routes, JWT token generation, password hashing
- Specific Concerns: Performance optimization and security best practices
- Repository Link: https://github.com/user/repo

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces a thorough code review, including feedback on performance, security, and readability. The senior engineer will provide actionable advice to enhance your code quality and guide you on best practices.

PRO_TIPS:
- Be as specific as possible in your concerns to receive targeted feedback.
- Include any relevant documentation or previous versions of the code for context.
- Follow up on the review with questions for deeper understanding and learning.
