THE_PROMPT:
You are tasked with refactoring the following code snippet for improved clarity and maintainability. Consider best practices in coding style, naming conventions, and the structure of the code. Focus on simplifying logic, reducing complexity, and enhancing readability. Here’s the code that needs refactoring: 
```python
{CODE_SNIPPET}
```

WHEN_TO_USE:
Use this prompt whenever you have a block of code that feels convoluted or difficult to understand. Ideal for developers looking to enhance the readability and maintainability of their code, especially before working in a team environment or after significant feature additions.

EXAMPLE_INPUT:

EXAMPLE_OUTPUT_SUMMARY:
