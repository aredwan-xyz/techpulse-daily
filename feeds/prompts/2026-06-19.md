PROMPT_TITLE: Complex Bug Diagnosis & Resolution

THE_PROMPT:
You are a debugging expert. I need your help to diagnose and fix a complex bug in my {PROGRAMMING_LANGUAGE} application. Here are the details of the issue: {BUG_DESCRIPTION}. The bug manifests when {CONDITIONS_OR_STEP_TO_REPRODUCE}. I have already tried the following solutions without success: {PREVIOUS_ATTEMPTS}. Please provide a detailed step-by-step analysis to pinpoint potential causes, suggestions for fixing the bug, and any resources that may help me understand this issue better.

WHEN_TO_USE:
Use this prompt when faced with challenging bugs that have proven resistant to standard debugging techniques. It is ideal for situations where you need a deeper analysis and structured guidance on how to approach a resolution.

EXAMPLE_INPUT:
- PROGRAMMING_LANGUAGE: Python
- BUG_DESCRIPTION: The application crashes with a segmentation fault.
- CONDITIONS_OR_STEP_TO_REPRODUCE: The crash occurs when processing large datasets.
- PREVIOUS_ATTEMPTS: I have checked memory usage and added try-except blocks, but the issue persists.

EXAMPLE_OUTPUT_SUMMARY:
This prompt generates a thorough examination of the bug, outlining possible underlying causes, offering practical troubleshooting steps, and recommending further learning materials. The output includes actionable insights tailored to the problem context.

PRO_TIPS:
- Be as specific as possible in the BUG_DESCRIPTION to aid accurate analysis.
- Include relevant code snippets or log file excerpts for more precise recommendations.
- Mention the tools or frameworks you are using to aid in the context of the debugging process.

WORKS_WITH: Claude · GPT-4 · Gemini
