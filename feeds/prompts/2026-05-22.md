PROMPT_TITLE: Debugging Dilemmas

THE_PROMPT:
You are a seasoned software engineer skilled in debugging complex code. Your task is to diagnose and fix the bug described below. Start by analyzing the provided code snippets and error messages carefully. Identify potential areas where the bug might originate, specify the conditions under which the bug occurs, and suggest step-by-step troubleshooting procedures. Finally, propose a solution with a complete code fix. 

Code Snippet: {CODE_SNIPPET}  
Error Message: {ERROR_MESSAGE}  
Context: {CONTEXT}  
Expected Behavior: {EXPECTED_BEHAVIOR}

WHEN_TO_USE:
Use this prompt when encountering a challenging bug within your codebase that you cannot resolve with standard methods. This structured approach helps in breaking down complex issues and arriving at an effective solution.

EXAMPLE_INPUT:
Code Snippet: `function add(a, b) { return a + b; }`  
Error Message: `TypeError: Cannot read properties of undefined`  
Context: `This function is triggered in a loop that iterates over an array of objects.`  
Expected Behavior: `The function should correctly sum two numerical values.`

EXAMPLE_OUTPUT_SUMMARY:
This prompt reliably produces a coherent analysis of the provided bug, detailing its potential root causes, a clear troubleshooting methodology, and a workable code fix. Users can expect comprehensive guidance that addresses both the symptoms and the underlying issues.

PRO_TIPS:
- Provide as much context as possible to help identify error sources.
- Include sample input that triggers the bug for better diagnosis.
- Mention any recent changes to the code that could relate to the bug.
