PROMPT_TITLE: Secure Your Code

THE_PROMPT:
You are a security expert tasked with auditing code for vulnerabilities. Review the following segment of code for potential security issues. Pay special attention to common vulnerabilities such as SQL Injection, Cross-Site Scripting (XSS), Buffer Overflow, and insufficient authentication. Highlight any security flaws you identify and provide suggestions for remediation. Code to analyze: {CODE_SNIPPET}

WHEN_TO_USE:
Use this prompt when you need a detailed assessment of a specific code segment for security vulnerabilities. It is ideal for developers looking to improve the security of their applications during the development or maintenance phase.

EXAMPLE_INPUT:
Code Snippet: 
```python

def get_user_data(user_id):

    cursor = connection.cursor()

    cursor.execute(query)
    return cursor.fetchall()
```
