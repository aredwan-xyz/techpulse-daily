PROMPT_TITLE: Comprehensive Security Code Review

THE_PROMPT:
You are an expert cybersecurity auditor tasked with reviewing the following code for security vulnerabilities. Your objective is to identify potential weaknesses and recommend best practices for mitigation. Please analyze the provided code snippet, focusing on common security issues such as SQL injection, cross-site scripting (XSS), and improper authentication methods. Additionally, highlight any areas that may expose sensitive data. Here’s the code snippet:
{CODE_SNIPPET}
After your analysis, provide a summary of the vulnerabilities found and suggest specific improvements for each identified issue.

WHEN_TO_USE:
Utilize this prompt when you need a thorough audit of a code snippet for potential security risks. It's ideal for developers seeking to enhance their codebase's security before deployment or during code reviews.

EXAMPLE_INPUT:
Here's the code snippet:
```
function authenticateUser($username, $password) {
    $query = "SELECT * FROM users WHERE username='$username' AND password='$password'";
    // Database execution logic here
}
```

EXAMPLE_OUTPUT_SUMMARY:
This prompt will generate a detailed report outlining the vulnerabilities present in the provided code. It will include identification of SQL injection risks, recommendations for prepared statements, and suggestions for secure password handling.

PRO_TIPS:
- Ensure your code snippet is as complete as possible to receive a comprehensive analysis.
- Specify any specific security standards or frameworks you’d like the audit to align with (e.g., OWASP).
- Consider including context about the application's environment (e.g., web, mobile) for more tailored recommendations.
