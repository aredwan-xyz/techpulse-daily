## What Is It?
SigNoz is an open-source observability platform that helps developers monitor and visualize their applications through real-time metrics, traces, and logs. It provides a comprehensive solution for troubleshooting and improving application performance.

## The Problem It Solves
Before SigNoz, developers relied heavily on a mix of disparate tools and manual log analysis to monitor application health and performance. This often led to slow debugging processes, incomplete visibility, and potential downtime. SigNoz addresses these pain points by offering an all-in-one solution that centralizes observability, allowing for faster issue resolution and better insights.

## Why It's Worth 10 Minutes of Your Time
1. **End-to-End Observability:** SigNoz combines metrics, traces, and logs in one platform, so you don't have to juggle multiple tools anymore. This holistic view simplifies debugging and enhances collaboration across teams.
2. **Open Source Flexibility:** As an open-source project, SigNoz allows you to customize and extend the platform to fit your unique requirements without vendor lock-in, helping you tailor observability to your specific tech stack.
3. **User-Friendly Interface:** The intuitive UI makes it easy for both developers and operations teams to navigate, visualize data, and create custom dashboards without steep learning curves.

## Quick Start
```bash
# Install SigNoz using Docker
docker run -d -p 3306:3306 --name signoz-mysql -e MYSQL_ROOT_PASSWORD=yourpassword -e MYSQL_DATABASE=signoz mysql:latest

# Run SigNoz backend and frontend
docker run -d -p 4200:4200 -v ./config.yml:/sigNoz/config/config.yml signoz/signoz:latest
```

## Real-World Example
```javascript
const { initTRPC } = require('@trpc/server');
  
const t = initTRPC.create();

t.procedure('getUser', {
  resolve: async ({ input }) => {
    // Start tracing
    const span = tracer.startSpan('getUserProcedure');
    try {
      const user = await fetchUserFromDatabase(input.id);
      return user;
    } finally {
      span.end(); // End the tracing span
    }
  },
});
```
