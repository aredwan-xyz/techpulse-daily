# Zellij

**Category:** CLI
**Link:** [https://github.com/zellij-org/zellij](https://github.com/zellij-org/zellij)

[![Repo](https://img.shields.io/badge/View_on_GitHub-181717?style=flat-square&logo=github)](
https://github.com/zellij-org/zellij)

---

## What Is It?
Zellij is a terminal workspace and multiplexer that enhances your command line experience by allowing you to split windows, manage multiple sessions, and collaborate with others—all within a single interface.

## The Problem It Solves
Before Zellij, developers often relied on terminal multiplexers like tmux or screen, which could be cumbersome and less user-friendly for newcomers. Zellij addresses this pain point with a more modern, intuitive interface that simplifies managing terminal sessions and providing features like layout persistence and collaboration.

## Why It's Worth 10 Minutes of Your Time
1. **Modern User Interface**: Zellij boasts a visually appealing and responsive UI that makes it easier to navigate and organize your terminal sessions compared to older tools.
2. **Collaboration Features**: It allows real-time collaborative sessions, enabling teams to work together in the same terminal, which is perfect for pair programming or training sessions.
3. **Flexible Layout Management**: With Zellij, you can create and save your custom layouts, making it easy to switch between different workflows without having to rearrange your workspace each time.

## Quick Start
```bash
# Install Zellij
cargo install zellij

# Start Zellij and create a new session
zellij
```

## Real-World Example
```bash
# Start a new Zellij session
zellij

# Split the terminal vertically
Ctrl + Shift + V

# Run a development server in one pane and monitor logs in another
cd my-app && npm start  # In one pane
tail -f app.log         # In another pane
```

## Who Should Use It
✅ Perfect for: Developers looking for a modern terminal experience, teams needing collaboration tools, or users of terminal multiplexers looking for a more intuitive alternative.  
⏩ Skip if: You prefer classic CLI tools without modern GUI enhancements or have minimal terminal work to do.

## The Honest Take
Zellij is a fresh and exciting alternative to traditional multiplexers, tailored for an enjoyable user experience and team collaboration. While it may require some adjustment for seasoned tmux users, its modern features and simplicity make it a standout choice in the realm of terminal tools.
