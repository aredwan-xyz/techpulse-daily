## 💡 Coding Challenge


TITLE: Find Median in Merged Sorted Arrays  
DIFFICULTY: Medium  
TOPIC: Sorting & Searching  
ESTIMATED_TIME: 30 minutes  

PROBLEM:  
You are given two sorted arrays of integers, `arr1` and `arr2`. Your task is to write a function that merges these arrays into one sorted array and finds the median of the merged array. The median is the middle value of the sorted data when the data set is of odd length, or the average of the two middle values when the data set is of even length. 

Implement the function `find_median(arr1: List[int], arr2: List[int]) -> float`.

EXAMPLES:  
Input:  
`arr1 = [1, 3]`  
`arr2 = [2]`  
Output:  
`2.0`  
Explanation: The merged array is `[1, 2, 3]`, and the median is `2.0`.

Input:  
`arr1 = [1, 2]`  
`arr2 = [3, 4]`  
Output:  
`2.5`  
Explanation: The merged array is `[1, 2, 3, 4]`, and the median is `(2 + 3) / 2` which equals `2.5`.  

CONSTRAINTS:  
- `0 <= len(arr1), len(arr2) <= 10^6`  
- Each element of the arrays is an integer within the range `[-10^6, 10^6]`  

HINT:  
Consider an efficient way to merge and find the median without explicitly combining the two arrays if possible.

SOLUTION_PYTHON:
```python
from typing import List

def find_median(arr1: List[int], arr2: List[int]) -> float:
    # Merger of two sorted arrays
    n1, n2 = len(arr1), len(arr2)
    merged = []
    i = j = 0

    while i < n1 and j < n2:
        if arr1[i] < arr2[j]:
            merged.append(arr1[i])
            i += 1
        else:
            merged.append(arr2[j])
            j += 1

    # Add remaining elements
    merged.extend(arr1[i:])
    merged.extend(arr2[j:])

    total_length = len(merged)
    mid = total_length // 2

    # Find median
    if total_length % 2 == 0:
        return (merged[mid - 1] + merged[mid]) / 2
    else:
        return merged[mid]

# Time Complexity: O(n1 + n2) where n1 and n2 are the lengths of the two arrays
# Space Complexity: O(n1 + n2) for the merged array
```

SOLUTION_JS:
```javascript
function findMedian(arr1, arr2) {
    const merged = [];
    let i = 0;
    let j = 0;

    while (i < arr1.length && j < arr2.length) {
        if (arr1[i] < arr2[j]) {
            merged.push(arr1[i]);
            i++;
        } else {
            merged.push(arr2[j]);
            j++;
        }
    }

    // Add remaining elements
    while (i < arr1.length) merged.push(arr1[i++]);
    while (j < arr2.length) merged.push(arr2[j++]);

    const totalLength = merged.length;
    const mid = Math.floor(totalLength / 2);

    if (totalLength % 2 === 0) {
        return (merged[mid - 1] + merged[mid]) / 2;
    } else {
        return merged[mid];
    }
}

// Time Complexity: O(n1 + n2) where n1 and n2 are the lengths of the two arrays
// Space Complexity: O(n1 + n2) for the merged array
```

TIME_COMPLEXITY: O(n1 + n2)  
SPACE_COMPLEXITY: O(n1 + n2)  

EXPLANATION:  
The solution involves merging two sorted arrays into one while maintaining their order, which allows for an efficient calculation of the median. This is done by iterating through both arrays and adding the smaller element to the merged list until one array is exhausted, followed by appending the remaining elements of either array. The median is then calculated based on whether the total length of the merged array is odd or even.

FOLLOW_UP:  
What optimization can you apply to find the median without fully merging the two arrays?

# Litestream

**Category:** Databases
**Link:** [https://github.com/benbjohnson/litestream](https://github.com/benbjohnson/litestream)

[![Repo](https://img.shields.io/badge/View_on_GitHub-181717?style=flat-square&logo=github)](
https://github.com/benbjohnson/litestream)

---

## What Is It?
Litestream is an open-source tool that provides continuous backup and replication for SQLite databases, allowing developers to automatically send changes to remote storage in real-time.

## The Problem It Solves
Before Litestream, developers using SQLite could struggle with the inherent limitations of local-only databases, especially regarding data safety and replication. Many relied on manual backup processes or scheduled tasks, which could lead to data loss or complexity in recovery operations. Litestream simplifies this by providing automated, real-time backups that improve data resilience and availability.

## Why It's Worth 10 Minutes of Your Time
1. **Real-Time Replication**: Litestream continually replicates your SQLite database, meaning your data is backed up instantly as changes occur, significantly reducing the risk of data loss compared to traditional, infrequent backups.
2. **Simplicity and Ease of Use**: Setting up Litestream is straightforward, with minimal configuration required, allowing you to integrate it into existing SQLite projects quickly without a steep learning curve.
3. **Cost-Effective Solution**: As an open-source tool, Litestream can save you money by replacing costly proprietary backup solutions while still providing the reliability and efficiency needed for critical applications.

## Quick Start
```bash
# Install Litestream
brew install litestream

# Initialize a Litestream configuration
litestream init

# Start Litestream to backup your SQLite database
litestream replicate
```

## Real-World Example
```go
import "github.com/benbjohnson/litestream"

// Connect to SQLite database and start streaming to S3
func main() {
   db := litestream.Open("my-database.sqlite")
   defer db.Close()

   // your application logic
   db.Exec("INSERT INTO users (name) VALUES ('Alice')")
}
```

## Who Should Use It
✅ Perfect for: Developers using SQLite for personal projects, prototyping, or lightweight apps that need simple backup solutions.  
⏩ Skip if: You're managing complex databases with high transaction rates or require advanced relational database features that SQLite does not support.

## The Honest Take
Litestream is a game-changer for anyone relying on SQLite, making data management significantly less stressful with its real-time backup capabilities. However, its use cases are somewhat limited to simpler applications, so it may not be the best fit for larger database needs. If you're looking for a hassle-free way to ensure your SQLite data is safe, Litestream is definitely worth trying.
