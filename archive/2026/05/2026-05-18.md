## 💡 Coding Challenge


TITLE: Count Distinct Set Bits
DIFFICULTY: Easy
TOPIC: Bit Manipulation
ESTIMATED_TIME: 20 minutes

PROBLEM:
Given an integer array `arr`, your task is to count the number of distinct set bits (1s) across all the integers in the array. Two integers can contribute the same bit position, but you should only count it once. 

For example, if an integer has a bit set at position 3 (which corresponds to the binary representation of 8), and another integer has a bit set at the same position, you should only count that bit once.

EXAMPLES:
Input: `arr = [3, 5, 7]`
Output: `4`
Explanation: The binary representations are:
- 3: 011
- 5: 101
- 7: 111
The distinct set bits across all integers are at positions 0, 1, 2, and 3, resulting in a total of 4 distinct set bits.

Input: `arr = [1, 2, 4]`
Output: `3`
Explanation: The binary representations are:
- 1: 001
- 2: 010
- 4: 100
The distinct set bits across all integers are at positions 0, 1, and 2, resulting in a total of 3 distinct set bits.

CONSTRAINTS:
- `1 ≤ arr.length ≤ 10^5`
- `0 ≤ arr[i] ≤ 10^6`

HINT:
Consider using a bitmask to track which bits have been encountered as you iterate through the array.

SOLUTION_PYTHON:
```python
def count_distinct_set_bits(arr):
    # Initialize a bitmask to track distinct bits
    bitmask = 0
    
    # Loop through each number in the array
    for num in arr:
        # Update bitmask with set bits from the current number
        bitmask |= num
    
    # Count the number of set bits in the bitmask
    return bin(bitmask).count('1')

# Example usage
print(count_distinct_set_bits([3, 5, 7]))  # Output: 4
print(count_distinct_set_bits([1, 2, 4]))  # Output: 3
```

SOLUTION_JS:
```javascript
function countDistinctSetBits(arr) {
    let bitmask = 0; // Initialize a bitmask to track distinct bits
    
    // Loop through each integer in the array
    for (let num of arr) {
        bitmask |= num; // Update the bitmask with the current number
    }

    // Count the number of set bits in the bitmask
    return bitmask.toString(2).split('0').join('').length;
}

// Example usage
console.log(countDistinctSetBits([3, 5, 7])); // Output: 4
console.log(countDistinctSetBits([1, 2, 4])); // Output: 3
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(1)

EXPLANATION:
This problem leverages bit manipulation by using a single bitmask to keep track of all distinct bits set across the integers in the array. As we iterate through each integer, we use the bitwise OR operation to update our bitmask, resulting in a streamlined solution that counts distinct bits in a single pass. This approach is optimal, making it efficient even for the upper limit of constraints.

FOLLOW_UP:
How would you modify the solution to return the specific bit positions (indices) that are set, rather than just the count of distinct set bits?

# LiteLLM

**Category:** AI/LLM
**Link:** [https://github.com/BerriAI/litellm](https://github.com/BerriAI/litellm)

[![Repo](https://img.shields.io/badge/View_on_GitHub-181717?style=flat-square&logo=github)](
https://github.com/BerriAI/litellm)

---

## What Is It?
LiteLLM is a lightweight library designed for building and running efficient large language models (LLMs) on limited hardware. It enables both developers and data scientists to leverage the power of text generation, completion, and understanding with a minimal resource footprint.

## The Problem It Solves
Before LiteLLM, many developers faced challenges in deploying large language models due to resource constraints, requiring cloud solutions with potentially high costs and latency. This tool alleviates such difficulties by optimizing model performance for local environments, allowing developers to experiment with LLMs without needing extensive infrastructure.

## Why It's Worth 10 Minutes of Your Time
1. **Lightweight Architecture**: LiteLLM is designed to run smoothly on consumer-grade hardware, making it accessible to more developers and encouraging experimentation without significant upfront investment.
2. **Ease of Use**: The library offers a straightforward API that allows you to start generating text with just a few lines of code, lowering the barrier for entry for those new to AI or LLMs.
3. **Customization**: You can easily customize your language models and tailor them to specific tasks, providing flexibility for a variety of applications, from chatbots to text analysis.

## Quick Start
```bash
pip install litellm
```

## Real-World Example
```python
from litellm import LiteLLM

# Initialize the LiteLLM model
model = LiteLLM(model_name='tiny-gpt')

# Generate text
output = model.generate(prompt="Once upon a time in a land far away,")
print(output)
```

## Who Should Use It
✅ Perfect for: Hobby developers, startups testing LLM applications, or anyone wanting to run LLMs on local machines without hefty cloud fees.  
⏩ Skip if: You need a very high-performing model for production workloads or have large-scale deployment requirements requiring extensive computational resources.

## The Honest Take
LiteLLM is a refreshing solution for those who want to tap into the power of language models without the complexities and costs of heavy infrastructure. While great for prototyping and small applications, it may fall short for enterprise-level tasks needing superior performance and scalability.
