## 💡 Coding Challenge


TITLE: Unique Character Counter
DIFFICULTY: Easy
TOPIC: Arrays & Strings
ESTIMATED_TIME: 20 minutes

PROBLEM:
You are tasked with creating a function that counts the number of unique characters in a given string. A unique character is defined as a character that appears only once in the input string. The function should return the count of these unique characters. Ensure that your function is case-insensitive, meaning 'A' and 'a' should be considered the same.

EXAMPLES:
Input: "Programming"
Output: 3
Explanation: The unique characters are 'P', 'r', and 'g', which occur only once when case is ignored.

Input: "Hello World"
Output: 7
Explanation: The unique characters are 'H', 'e', 'W', 'r', 'l', and 'd', totaling 7 unique occurrences.

CONSTRAINTS:
- The input string will only contain printable ASCII characters.
- The length of the input string will not exceed 1000 characters.

HINT:
Consider using a data structure that allows you to easily count occurrences of characters.

SOLUTION_PYTHON:
```python
def unique_character_count(s: str) -> int:
    # Use a dictionary to count occurrences of each character
    char_count = {}
    
    for char in s.lower():  # Convert to lower case for case insensitivity
        if char in char_count:
            char_count[char] += 1
        else:
            char_count[char] = 1
            
    # Count how many characters occur only once
    unique_count = sum(1 for count in char_count.values() if count == 1)
    
    return unique_count

# Time Complexity: O(n), where n is the length of the string
# Space Complexity: O(k), where k is the number of unique characters
```

SOLUTION_JS:
```javascript
function uniqueCharacterCount(s) {
    // Object to count occurrences of each character
    const charCount = {};
    
    // Loop through the string in lowercase
    for (let char of s.toLowerCase()) {
        charCount[char] = (charCount[char] || 0) + 1;
    }
    
    // Count characters that occur only once
    let uniqueCount = 0;
    for (let count of Object.values(charCount)) {
        if (count === 1) {
            uniqueCount++;
        }
    }
    
    return uniqueCount;
}

// Time Complexity: O(n), where n is the length of the string
// Space Complexity: O(k), where k is the number of unique characters
```

TIME_COMPLEXITY: O(n)
SPACE_COMPLEXITY: O(k)

EXPLANATION:
The approach utilizes a hash table (dictionary in Python, and an object in JavaScript) to count the occurrences of each character in the string, processing each character in a single pass. After counting, a second pass is made to sum the characters that appear only once. This method is efficient with respect to both time and space, as it iterates through the string a constant number of times.

FOLLOW_UP:
How would your solution change if you were required to return a list of the unique characters instead of just the count?

# Encore

**Category:** Backend
**Link:** [https://github.com/encoredev/encore](https://github.com/encoredev/encore)

[![Repo](https://img.shields.io/badge/View_on_GitHub-181717?style=flat-square&logo=github)](
https://github.com/encoredev/encore)

---

## What Is It?
Encore is an open-source backend development framework that simplifies building and deploying serverless applications in Go. It empowers developers to focus on writing business logic rather than managing infrastructure.

## The Problem It Solves
Before Encore, backend developers often grappled with complex server and infrastructure setups, spending significant time configuring environments and deploying code. Encore eliminates these pain points by enabling a straightforward serverless model, allowing developers to get their applications up and running with minimal friction.

## Why It's Worth 10 Minutes of Your Time
1. **Rapid Development**: Encore provides built-in features like authentication, database integration, and API routing out of the box, enabling you to spin up a new backend in minutes.
2. **Seamless Deployment**: The framework simplifies deployment to major cloud providers, automatically handling scaling and infrastructure concerns so you can focus solely on code.
3. **Powerful Dev Experience**: With its out-of-the-box support for local development, you can test and debug your entire backend application on your machine without any external dependencies.

## Quick Start
```bash
# Install Encore
go install encore.dev/encore@latest

# Create a new Encore project
encore init myapp

# Run the local development server
cd myapp && encore run
```

## Real-World Example
```go
// Define an API endpoint in your Encore app
package main

import (
    "encore.dev/encore"
    "net/http"
)

func main() {
    encore.Route("/hello").Methods(http.MethodGet)(helloHandler)
}

func helloHandler(w http.ResponseWriter, r *http.Request) {
    w.Write([]byte("Hello, Encore!"))
}
```

## Who Should Use It
✅ Perfect for: Developers looking to build serverless applications quickly, teams needing to streamline backend infrastructure, or those looking to enhance collaboration in Go projects.  
⏩ Skip if: You're working on large-scale applications that require fine-grained control over the infrastructure or prefer traditional deployment models.

## The Honest Take
Encore is a powerful tool that significantly simplifies backend development while prioritizing developer experience. However, its serverless nature may not appeal to those who need more control over their infrastructure. Bottom line: if you're looking for speed and simplicity in building serverless applications, give Encore a try!
